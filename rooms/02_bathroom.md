# ห้องน้ำ (Bathroom)

## บรรยากาศ
กระเบื้องสีขาวที่เคยสะอาดตอนนี้มีรอยร้าวเป็นเส้นบางๆ น้ำในอ่างล้างหน้ายังขังอยู่สีน้ำตาลอ่อน หลอดไฟกะพริบเป็นจังหวะ กระจกบานใหญ่เหนืออ่างล้างหน้ามีไอน้ำเกาะ — แม้ว่าจะยังไม่มีใครอาบน้ำ

## เงื่อนไขการเข้า
- ต้องมี **ลูกบิดประตูห้องน้ำ** จากใต้เตียงห้องนอน

## วัตถุที่โต้ตอบได้

| วัตถุ | การกระทำ | ผล |
|------|----------|-----|
| **กระจก** | มอง | ภาพสะท้อนไม่ตรงกับการเคลื่อนไหว |
| **กระจก (ไฟดับ)** | ดับไฟแล้วมอง | เห็นรหัสตู้เสื้อผ้าห้องนอน เขียนด้วยหมึกเรืองแสง |
| **อ่างล้างหน้า** | เปิดก๊อก | น้ำสีแดงออกมา 3 วินาที แล้วใสตามปกติ |
| **ตู้ยา** | เปิด (ต้องมีกุญแจเล็ก) | เห็นขวดยาหลายขวด |
| **ขวดยา** | ตรวจสอบเลขล็อต | บันทึก **"เลขหลักที่ 2"** ของรหัสรั้วลงสมุด |
| **ฝักบัว** | เปิด | ต้องเปิดน้ำเย็นก่อนน้ำร้อนเสมอ |
| **สวิตช์ไฟ** | ปิดไฟ | ห้องมืด — กระจกแสดงรหัสซ่อน |
| **พื้น** | ตรวจสอบรอยแตก | พบ **เศษกระดาษ** — คำใบ้เรื่องลำดับยา |

## ปริศนา

### ปริศนา A — กระจกรหัสซ่อน
- **วิธี:** ปิดไฟในห้องน้ำ → มองกระจก
- **รหัสปรากฏ:** ตัวเลข 3 หลักสำหรับตู้เสื้อผ้าห้องนอน
- **หมายเหตุ:** หากมองกระจกนานเกินไปโดยไม่ปิดไฟก่อน → **ตาย**

### ปริศนา B — ตู้ยา (ต้องใช้กุญแจเล็ก)
- **ขวดยาในตู้:** 5 ขวด มีเพียง 1 ขวดที่ถูกต้อง
- **เบาะแส:** เศษกระดาษบนพื้น + รูปในสมุดบันทึกเก่าห้องนอน
- **ขวดที่ถูก:** มีเลขล็อตเป็นตัวเลขหลักที่ 2 ของรหัสรั้ว
- **ผล:** นำขวดยาออกมา → ด้านหลังตู้มีกุญแจเล็กสำหรับห้องน้ำ → **ปลดล็อคโถงทางเดิน** (ผ่านปริศนาในโถง)

### ปริศนา C — ฝักบัว (ลำดับน้ำ)
- **กฎ:** น้ำเย็นก่อนเสมอ
- **ถ้าเปิดน้ำร้อนก่อน:** ไอน้ำเดือดออกมา → **ตาย**
- **รางวัล:** ไม่มีไอเทม แต่ห้องน้ำเปลี่ยนสถานะเป็น "ทำความสะอาดแล้ว" ซึ่งปลดล็อคบางอย่างในห้องครัว

## เงื่อนไขการตาย

| การกระทำ | สาเหตุการตาย |
|----------|-------------|
| มองกระจกนานเกิน 8 วินาที (ไฟเปิด) | ภาพสะท้อนเดินออกมาจากกระจก |
| เปิดน้ำร้อนก่อนน้ำเย็น | ไอน้ำเดือดระเบิดออกมา |
| กินยาผิดขวด | ชักกระตุกและเสียชีวิต |
| เปิดตู้ยาโดยไม่มีกุญแจ (งัด) | ตู้ล็อคแน่น — มือติดค้าง มีอะไรดึงจากข้างใน |

## สิ่งที่เปลี่ยนเมื่อกลับมา

- ครั้งที่ 2: รอยเขียนบนกระจกเพิ่มขึ้น — ข้อความเตือน
- ครั้งที่ 3+: ตู้ยาเปิดอยู่เอง ขวดยาบางขวดหายไป

## Backtracking จากห้องอื่น

ห้องนี้ไม่ต้องการการ backtrack จากห้องอื่น
แต่ **ต้องมาห้องนี้ก่อน** เพื่อเอารหัสกลับไปห้องนอน

## ไอเทมที่ได้จากห้องนี้

- `รหัสตู้เสื้อผ้า` (บันทึกในสมุด) → ใช้ที่ห้องนอน
- `เลขรหัสหลักที่ 2` (บันทึกในสมุด) → ใช้ที่รั้ว
