# ห้องน้ำ (Bathroom)

## วัตถุประสงค์
เตรียมตัวให้เสร็จเรียบร้อย เพื่อเริ่มต้นวันใหม่อย่างราบรื่น

## บรรยากาศ
กระเบื้องสีขาวชื้นแฉะ ขวดสบู่ล้มหกอยู่ใกล้ประตูทางเข้า สบู่เหลวกำลังไหลออกมาเป็นแอ่งบนพื้น หลอดไฟเพดานสว่างอยู่แต่เริ่มกะพริบติดๆ ดับๆ เป็นจังหวะ ไดร์เป่าผมเสียบปลั๊กอยู่ตกนอนบนพื้นใกล้อ่างอาบน้ำ ตู้กระจกเหนืออ่างล้างหน้าปิดอยู่ อ่างอาบน้ำว่างเปล่ารอให้เติมน้ำลงไป

## สถานะเริ่มต้น
- เข้าได้จากห้องนอน (ผ่าน win flow ห้องนอนแล้ว)
- **ขวดสบู่:** ล้มหกอยู่ สบู่กำลังไหลออกมาเรื่อยๆ (ต้องรีบจัดการ)
- **ไฟห้องน้ำ:** สว่างอยู่ แต่เริ่มกะพริบช้าๆ (จะรุนแรงขึ้นตามเวลา)
- **ไดร์เป่าผม:** เสียบปลั๊กอยู่ ตกอยู่บนพื้น (อันตราย)
- **ประตูห้องนอนออกสู่โถง:** ล็อคอยู่ (จะพบกุญแจหลังระบายน้ำจากอ่าง)

## ความผิดปกติในห้อง (Anomalies)

| ความผิดปกติ | พฤติกรรม | สถานะ |
|------------|----------|-------|
| **ขวดสบู่หก** | ล้มอยู่ใกล้ประตู สบู่เหลวกำลังไหลออกมาเป็นแอ่งบนพื้น | ต้องรีบเก็บตั้งขึ้นก่อนสบู่จะไหลออกมากเกินไป |
| **ตู้ยาและวิตามิน** | ตู้กระจกเหนืออ่างล้างหน้า ข้างในมียาและวิตามินหลายชนิด | ต้องเปิดตู้ → เลือกทานยาสีชมพูเข้ม |
| **ไฟห้องน้ำ** | สว่างอยู่แต่เริ่มกะพริบติดๆ ดับๆ ช้าๆ → จะถี่ขึ้นเรื่อยๆ ตามเวลา | จะหยุดกะพริบหลังทานยาสีชมพูเข้ม |
| **ไดร์เป่าผม** | เสียบปลั๊กอยู่ ตกนอนบนพื้นใกล้อ่างอาบน้ำ | ต้องถอดปลั๊กก่อน → แล้วเก็บเข้าตู้ |
| **อ่างอาบน้ำ** | ว่างเปล่า มีก๊อกน้ำร้อน-เย็นให้ปรับ | ต้องเปิดน้ำ → ปรับสลับร้อน-เย็น → ปิดเมื่อเต็ม → แช่ → ระบายน้ำ |
| **กุญแจห้องนอน** | ซ่อนอยู่ก้นอ่างอาบน้ำ (มองไม่เห็นเมื่อมีน้ำ) | จะพบหลังระบายน้ำออกจากอ่าง |

## วัตถุที่โต้ตอบได้

| วัตถุ | การกระทำ | เงื่อนไข | ผล |
|------|----------|----------|-----|
| **ขวดสบู่** | เก็บตั้งขึ้น | — | สบู่หยุดไหล พื้นยังลื่นเล็กน้อยแต่ปลอดภัย |
| **พื้นบริเวณสบู่หก** | เดินผ่าน | สบู่ไหลออกมาเยอะแล้ว (ไม่เก็บขวดทัน) | ☠️ ลื่นล้ม → ตาย |
| **ตู้ยา** | เปิด | — | เห็นยาและวิตามินหลายชนิดเรียงอยู่ |
| **ยาสีชมพูเข้ม** | ทาน | เปิดตู้ยาแล้ว | ✅ ไฟห้องน้ำหยุดกะพริบ → สภาพจิตดีขึ้น |
| **ยาชนิดอื่น (สีผิด)** | ทาน | เปิดตู้ยาแล้ว | ☠️ ทานยาผิด → ชักกระตุกเสียชีวิต |
| **วิตามิน** | ทาน | เปิดตู้ยาแล้ว | 🩹 วิตามินติดคอ → ได้รับบาดเจ็บ (ไม่ตาย) |
| **ไดร์เป่าผม (เสียบปลั๊กอยู่)** | ถอดปลั๊ก | — | ไดร์หมดไฟ ปลอดภัยที่จะหยิบ |
| **ไดร์เป่าผม (ถอดปลั๊กแล้ว)** | เก็บเข้าตู้ | ถอดปลั๊กแล้ว | เก็บเรียบร้อย ไม่เป็นอันตราย |
| **ไดร์เป่าผม (เสียบปลั๊กอยู่)** | หยิบ / เดินผ่าน | ยังเสียบปลั๊ก + ตัวเปียก (หลังแช่น้ำ) | ☠️ ไฟดูด → ตาย |
| **ก๊อกน้ำอ่างอาบน้ำ** | เปิดน้ำ | ถอดปลั๊กไดร์แล้ว + เก็บไดร์แล้ว | น้ำเริ่มไหลเข้าอ่าง |
| **ก๊อกน้ำ (ปรับอุณหภูมิ)** | สลับร้อน-เย็น | น้ำกำลังไหล | ต้องคอยสลับสม่ำเสมอ ไม่ปล่อยร้อนหรือเย็นนานเกิน |
| **ก๊อกน้ำ** | ปิดน้ำ | น้ำเต็มอ่างแล้ว | น้ำหยุดไหล ระดับพอดี |
| **อ่างอาบน้ำ (น้ำเต็ม)** | ลงแช่ | ปิดน้ำเมื่อเต็มอ่างแล้ว | เริ่มนับเวลาแช่ |
| **อ่างอาบน้ำ (แช่ครบเวลา)** | ขึ้นจากอ่าง | แช่ครบเวลาแล้ว | ตัวละครขึ้นจากอ่าง |
| **จุกระบายน้ำ** | ดึงเปิด | ขึ้นจากอ่างแล้ว | น้ำระบายออก → พบกุญแจห้องนอนที่ก้นอ่าง |
| **กุญแจห้องนอน** | หยิบ | น้ำระบายหมดแล้ว | ได้กุญแจ → ปลดล็อคประตูห้องนอนออกสู่โถงทางเดิน |
| **ประตูห้องนอน (ออกสู่โถง)** | เปิด | มีกุญแจห้องนอนแล้ว | ✅ ออกจากห้องนอนไปโถงทางเดิน → 🏁 ผ่านด่าน |

## Win Flow (ลำดับการเล่นที่ถูกต้อง)

```
① รีบเก็บขวดสบู่ตั้งขึ้น (ก่อนสบู่ไหลออกมาเพิ่ม)
    │
② เปิดตู้ยาบริเวณอ่างล้างหน้า
    │
③ เลือกทานยาสีชมพูเข้ม
    │
    └─→ ไฟห้องน้ำหยุดกะพริบ
    │
④ ถอดปลั๊กไดร์เป่าผม
    │
⑤ เก็บไดร์เป่าผมเข้าตู้
    │
⑥ เปิดน้ำใส่อ่างอาบน้ำ
    │
⑦ คอยสลับร้อน-เย็นให้อุณหภูมิพอดี
    │
⑧ ปิดน้ำเมื่อเต็มอ่าง
    │
⑨ ลงไปแช่น้ำในอ่าง
    │
⑩ รอจนแช่ครบเวลา → ขึ้นจากอ่าง
    │
⑪ ดึงจุกระบายน้ำจากอ่าง
    │
    └─→ พบกุญแจห้องนอนที่ก้นอ่าง
    │
⑫ หยิบกุญแจ → ปลดล็อคประตูห้องนอนออกสู่โถงทางเดิน
    │
⑬ เปิดประตูออกไป → 🏁 ผ่านด่าน
```

## เบาะแส

| แหล่ง | เนื้อหา |
|-------|---------|
| **กระดาษโน้ตจากห้องนอน** | _"อย่าลืมทานยาเม็ดสีชมพูเข้มนะ"_ → คำใบ้ว่าต้องเลือกยาสีชมพูเข้ม |
| **กุญแจก้นอ่างอาบน้ำ** | พบหลังจากแช่น้ำในอ่างและระบายน้ำทิ้ง → กุญแจเปิดประตูห้องนอนออกสู่โถงทางเดิน |

## เงื่อนไขการตาย ☠️

| สถานการณ์ | เงื่อนไข | สาเหตุการตาย |
|-----------|----------|-------------|
| **ลื่นล้ม** | เดินผ่านบริเวณสบู่ที่หกออกมาเยอะ (ไม่เก็บขวดสบู่ทัน) | พื้นลื่นจากสบู่ที่ไหลออกมามาก ลื่นล้มหัวฟาดพื้น → ตาย |
| **ทานยาผิด** | เลือกทานยาที่ไม่ใช่สีชมพูเข้ม | ยาผิดชนิด ร่างกายชักกระตุก → ตาย |
| **ช็อคจากไฟกะพริบ** | ไม่ทานยาสีชมพูเข้มภายในเวลาที่กำหนด | ไฟกะพริบรุนแรงและถี่ขึ้นเรื่อยๆ → ตัวละครเกิดอาการหลอน panic ช็อค → ตาย |
| **น้ำร้อนเกิน** | ปล่อยก๊อกน้ำร้อนนานเกินไป ไม่สลับเย็น | แช่น้ำในอุณหภูมิที่ร้อนเกินไป ผิวหนังไหม้ → ตาย |
| **น้ำเย็นเกิน** | ปล่อยก๊อกน้ำเย็นนานเกินไป ไม่สลับร้อน | แช่น้ำในอุณหภูมิที่เย็นเกินไป ร่างกายอุณหภูมิต่ำ → ตาย |
| **น้ำล้นอ่าง** | ไม่ปิดน้ำเมื่อเต็มอ่าง ปล่อยให้ไหลล้นออกมา | น้ำไหลท่วมพื้น ไหลไปโดนไดร์เป่าผมที่เสียบปลั๊กอยู่ (ถ้ายังไม่ถอดปลั๊ก) → ไฟดูดตาย / น้ำท่วมพื้นจำนวนมาก → ลื่นล้ม |
| **ไฟดูด (ไดร์ตอนเปียก)** | เดินผ่านหรือจับไดร์เป่าผมที่เสียบปลั๊กอยู่ หลังจากแช่น้ำเสร็จ (ตัวเปียก) | ไฟดูดจากไดร์ที่ยังเสียบปลั๊ก + ตัวเปียก → ตาย |
