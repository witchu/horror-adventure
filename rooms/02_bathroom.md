# ห้องน้ำ (Bathroom)

## วัตถุประสงค์
เตรียมตัวให้เสร็จเรียบร้อย เพื่อเริ่มต้นวันใหม่อย่างราบรื่น

## บรรยากาศ
กระเบื้องสีขาวชื้นแฉะ ขวดสบู่ล้มหกอยู่ใกล้ประตูทางเข้า สบู่เหลวกำลังไหลออกมาเป็นแอ่งบนพื้น หลอดไฟเพดานสว่างอยู่แต่เริ่มกะพริบติดๆ ดับๆ เป็นจังหวะ ไดร์เป่าผมเสียบปลั๊กอยู่ตกนอนบนพื้นใกล้อ่างอาบน้ำ ตู้กระจกเหนืออ่างล้างหน้าปิดอยู่ อ่างอาบน้ำว่างเปล่ารอให้เติมน้ำลงไป

## สถานะเริ่มต้น
- เข้าได้จากห้องนอน (ผ่าน win flow ห้องนอนแล้ว)
- **ขวดสบู่:** ล้มหกอยู่ สบู่กำลังไหลออกมาเรื่อยๆ (ต้องรีบจัดการ)
- **ไฟห้องน้ำ:** สว่างอยู่ แต่เริ่มกะพริบช้าๆ (จะรุนแรงขึ้นตามเวลา)
- **ไดร์เป่าผม:** เสียบปลั๊กอยู่ ตกอยู่บนพื้น (อันตราย)
- **ประตูห้องนอนออกสู่โถง:** ล็อคอยู่ (จะพบกุญแจหลังระบายน้ำจากอ่าง)

## ความผิดปกติในห้อง (Anomalies)

| ความผิดปกติ | พฤติกรรม | สถานะ |
|------------|----------|-------|
| **ขวดสบู่หก** | ล้มอยู่ใกล้ประตู สบู่เหลวกำลังไหลออกมาเป็นแอ่งบนพื้น | ต้องรีบเก็บตั้งขึ้นก่อนสบู่จะไหลออกมากเกินไป |
| **ตู้ยาและวิตามิน** | ตู้กระจกเหนืออ่างล้างหน้า เมื่อเปิดตู้จะมีหน้าต่างตัวเลือกยา 6 กระปุก | ต้องเปิดตู้ → เลือกระบุหมายเลขกระปุกที่ถูกต้อง (กระปุกที่ 1) |
| **ไฟห้องน้ำ** | สว่างอยู่แต่เริ่มกะพริบติดๆ ดับๆ ช้าๆ → จะถี่ขึ้นเรื่อยๆ ตามเวลา | จะหยุดกะพริบหลังทานยาจากกระปุกที่ 1 |
| **ไดร์เป่าผม** | เสียบปลั๊กอยู่ ตกนอนบนพื้นใกล้อ่างอาบน้ำ | ต้องถอดปลั๊กก่อน → แล้วเก็บเข้าตู้ |
| **อ่างอาบน้ำ** | ว่างเปล่า มีก๊อกน้ำร้อน-เย็นให้ปรับ | ต้องเปิดน้ำ → ปรับสลับร้อน-เย็น → ปิดเมื่อเต็ม → แช่ → ระบายน้ำ |
| **กุญแจห้องนอน** | ติดค้างอยู่ลึกในท่อระบายน้ำของอ่างอาบน้ำ | จะถูกดันขึ้นมาให้พบได้ก็ต่อเมื่อเติมน้ำแช่แล้วดึงจุกระบายน้ำทิ้งเท่านั้น |

## วัตถุที่โต้ตอบได้

### ตัวเลือกหน้าต่างตู้ยา (Medicine Cabinet UI)
เมื่อผู้เล่นโต้ตอบ "เปิด" ตู้ยา จะมีหน้าต่างขึ้นมาให้ผู้เล่นเลือกยา 6 กระปุก โดยตัวเลือก**จะแสดงแค่กระปุกที่ และลักษณะของชนิดยา**เท่านั้น (ไม่บอกชื่อยา):
- **กระปุกที่ 1 :** เม็ดเคลือบ สีชมพูเข้ม *(Dev Note: ยาลดกังวล - ✅ ถูกต้อง/Win)*
- **กระปุกที่ 2 :** แคปซูล สีเหลืองสด *(Dev Note: วิตามิน B - 🩹 บาดเจ็บ)*
- **กระปุกที่ 3 :** ยาชนิดน้ำ สีฟ้า *(Dev Note: แอลกอฮอล์ล้างแผล - ☠️ ตาย)*
- **กระปุกที่ 4 :** แคปซูล สีดำ *(Dev Note: ไซยาไนด์ - ☠️ ตาย)*
- **กระปุกที่ 5 :** เม็ดใหญ่ทรงรี สีส้ม *(Dev Note: วิตามิน C - 🩹 บาดเจ็บ)*
- **กระปุกที่ 6 :** ยาชนิดน้ำ สีน้ำตาลเข้ม *(Dev Note: เบตาดีน - ☠️ ตาย)*

### ระบบก๊อกน้ำอ่างอาบน้ำ (Bathtub Faucet UI)
การโต้ตอบเพื่อเติมน้ำลงในอ่างอาบน้ำจะมีระบบสถานะและการแสดงผลดังนี้:
- **UI สถานะน้ำ:** หน้าจอจะแสดงเกจบอกปริมาตรน้ำรวมในอ่าง (เต็มที่ 100%) พร้อมกับระบุสัดส่วนอุณหภูมิว่าขณะนั้นประกอบด้วย **น้ำร้อนอยู่กี่ %** และ **น้ำเย็นอยู่กี่ %**
- **การเริ่มเปิดน้ำ:** หากผู้เล่นคลิกที่ก๊อกน้ำครั้งแรก หรือคลิกในช่วงที่ก๊อกปิดอยู่ น้ำที่จะไหลออกมาเริ่มต้นจะเป็นกระแส **"น้ำร้อน"** เสมอ
- **การโต้ตอบขณะไหล:** เมื่อผู้เล่นคลิกที่ก๊อกในขณะที่น้ำกำลังไหลอยู่ จะมีหน้าต่างตัวเลือกปรากฏขึ้นมาเพื่อจัดการก๊อกน้ำ คือ **`[ น้ำเย็น ] / [ ปิด ] / [ น้ำร้อน ]`**

| วัตถุ | การกระทำ | เงื่อนไข | ผล |
|------|----------|----------|-----|
| **ขวดสบู่** | เก็บตั้งขึ้น | — | สบู่หยุดไหล พื้นยังลื่นเล็กน้อยแต่ปลอดภัย |
| **พื้นบริเวณสบู่หก** | เดินผ่าน | สบู่ไหลออกมาเยอะแล้ว (ไม่เก็บขวดทัน) | ☠️ ลื่นล้ม → ตาย |
| **ตู้ยา** | เปิด | — | ปรากฏหน้าต่างตัวเลือกกระปุกยา 6 กระปุกให้ผู้เล่นเลือก |
| **กระปุกที่ 1** | เลือกทาน | เปิดตู้ยาแล้ว | ✅ ไฟห้องน้ำหยุดกะพริบ → สภาพจิตดีขึ้น |
| **กระปุกที่ 2, 5** | เลือกทาน | เปิดตู้ยาแล้ว | 🩹 ได้รับบาดเจ็บ (เสีย HP แต่ไม่ตาย) |
| **กระปุกที่ 3, 4, 6** | เลือกทาน | เปิดตู้ยาแล้ว | ☠️ ทานยาอันตรายร้ายแรง → ตายทันที |
| **ไดร์เป่าผม (เสียบปลั๊กอยู่)** | ถอดปลั๊ก | — | ไดร์หมดไฟ ปลอดภัยที่จะหยิบ |
| **ไดร์เป่าผม (ถอดปลั๊กแล้ว)** | เก็บเข้าตู้ | ถอดปลั๊กแล้ว | เก็บเรียบร้อย ไม่เป็นอันตราย |
| **ไดร์เป่าผม (เสียบปลั๊กอยู่)** | หยิบ / เดินผ่าน | ยังเสียบปลั๊ก + ตัวเปียก (หลังแช่น้ำ) | ☠️ ไฟดูด → ตาย |
| **ก๊อกน้ำอ่างอาบน้ำ (ขณะปิดอยู่)** | คลิกเพื่อเปิด | ถอดปลั๊กและเก็บไดร์เป่าผมแล้ว | เริ่มปล่อย **น้ำร้อน** ลงอ่าง พร้อมแสดง UI หน้าต่างสถานะน้ำปริมาตรรวม, % น้ำร้อน, % น้ำเย็น |
| **ก๊อกน้ำอ่างอาบน้ำ (ขณะน้ำไหล)** | คลิกสลับระบบ | น้ำกำลังไหล | แสดงตัวเลือก `[ น้ำเย็น ]` / `[ ปิด ]` / `[ น้ำร้อน ]` สำหรับสลับระบบน้ำ |
| **คำสั่ง "ปิด" ก๊อกน้ำ** | เลือกคลิก "ปิด" | — | น้ำหยุดไหลและรักษาสัดส่วน % เพื่อเตรียมพร้อมให้ตัวละครลงแช่ (หากเปิดใหม่จะถูกรีเซ็ตไปที่น้ำร้อน) |
| **อ่างอาบน้ำ (น้ำเต็ม)** | ลงแช่ | ปิดน้ำเมื่อเต็มอ่างแล้ว | เริ่มนับเวลาแช่ |
| **อ่างอาบน้ำ (แช่ครบเวลา)** | ขึ้นจากอ่าง | แช่ครบเวลาแล้ว | ตัวละครขึ้นจากอ่าง (สภาพตัวเปียกชุ่ม) |
| **ผ้าเช็ดตัว** | นำมาเช็ดตัว | ขึ้นจากอ่างแล้ว + มีผ้าเช็ดตัวใน Inventory | ตัวละครเช็ดตัวจนแห้งสนิท ป้องกันอันตรายจากไฟดูดได้ 100% |
| **จุกระบายน้ำ** | ดึงเปิด | ขึ้นจากอ่างแล้ว | น้ำแรงดันสูงระบายออก ทำให้น้ำดันกุญแจห้องนอนหลุดลอยขึ้นมาจากท่อระบาย |
| **กุญแจห้องนอน** | หยิบ | น้ำระบายหมดและกุญแจหลุดออกมาแล้ว | ได้กุญแจไขประตูออกสู่โถง → เตรียมกดกลับไปยังห้องนอน |

## Win Flow (ลำดับการเล่นที่ถูกต้อง)

```
① รีบเก็บขวดสบู่ตั้งขึ้น (ก่อนสบู่ไหลออกมาเพิ่ม)
    │
② เปิดตู้ยาบริเวณอ่างล้างหน้า (จะปรากฏหน้าต่างตัวเลือกยา)
    │
③ เลือกทาน "กระปุกที่ 1 : เม็ดเคลือบ สีชมพูเข้ม"
    │
    └─→ ไฟห้องน้ำหยุดกะพริบ
    │
④ ถอดปลั๊กไดร์เป่าผม
    │
⑤ เก็บไดร์เป่าผมเข้าตู้
    │
⑥ คลิกก๊อกน้ำเพื่อเปิดน้ำใส่อ่างอาบน้ำ (จะเริ่มไหลที่น้ำร้อนเสมอ)
    │
⑦ คลิกก๊อกน้ำขณะไหลเพื่อใช้ตัวเลือกสลับ "น้ำเย็น" และ "น้ำร้อน" ดูแลอัตราส่วนอุณหภูมิไม่ให้มีตัวใดตัวหนึ่ง > 80%
    │
⑧ เลือก "ปิด" น้ำเมื่อระดับน้ำรวมเต็มอ่าง 100%
    │
⑨ ลงไปแช่น้ำในอ่าง
    │
⑩ รอจนแช่ครบเวลา → ขึ้นจากอ่าง (สภาพตัวเปียก)
    │
    ├─→ นำผ้าเช็ดตัวจาก Inventory ออกมาเช็ดตัวให้แห้ง (เพื่อป้องกันตายจากไฟดูด)
    │
⑪ ดึงจุกระบายน้ำจากอ่าง
    │
    └─→ น้ำที่ไหลทิ้งจะผลักดันกุญแจห้องนอนให้หลุดลอยขึ้นมาจากท่อระบาย
    │
⑫ หยิบกุญแจห้องนอน
    │
⑬ กดเดินย้อนกลับเข้า "ห้องนอน" → ใช้กุญแจไขประตูออกสู่โถงทางเดิน → 🏁 ผ่านด่าน
```

## เบาะแส

| แหล่ง | เนื้อหา |
|-------|---------|
| **กระดาษโน้ตจากห้องนอน** | _"อย่าลืมทานยาเม็ดสีชมพูเข้มนะ"_ → คำใบ้ให้ผู้เล่นรอดตายโดยการเลือกทาน **กระปุกที่ 1 : เม็ดเคลือบ สีชมพูเข้ม** |
| **กุญแจก้นอ่างอาบน้ำ** | พบหลังจากแช่น้ำในอ่างและระบายน้ำทิ้ง → กุญแจเปิดประตูห้องนอนออกสู่โถงทางเดิน |

## เงื่อนไขการตาย ☠️

| สถานการณ์ | เงื่อนไข | สาเหตุการตาย |
|-----------|----------|-------------|
| **ลื่นล้ม** | เดินผ่านบริเวณสบู่ที่หกออกมาเยอะ (ไม่เก็บขวดสบู่ทัน) | พื้นลื่นจากสบู่ที่ไหลออกมามาก ลื่นล้มหัวฟาดพื้น → ตาย |
| **ทานยาพิษ/เคมี** | เลือกทาน "กระปุกที่ 3 (น้ำสีฟ้า)", "กระปุกที่ 4 (แคปซูลดำ)", หรือ "กระปุกที่ 6 (น้ำสีน้ำตาลเข้ม)" | สารเคมีหรือพิษทำลายระบบภายในร่างกายอย่างรุนแรง → ตาย |
| **ช็อคจากไฟกะพริบ** | ไม่ทานยากระปุกที่ 1 ภายในเวลาที่กำหนด | ไฟกะพริบรุนแรงและถี่ขึ้นเรื่อยๆ → ตัวละครเกิดอาการหลอน panic ช็อค → ตาย |
| **น้ำร้อนเกิน** | ใส่น้ำร้อนให้มีสัดส่วนมากกว่า 80% ขึ้นไป เมื่อลงแช่ตอนน้ำเต็มอ่าง | แช่น้ำในอุณหภูมิที่ร้อนจัด ผิวหนังพุพองถูกลวกอย่างรุนแรง → ตาย |
| **น้ำเย็นเกิน** | ใส่น้ำเย็นให้มีสัดส่วนมากกว่า 80% ขึ้นไป เมื่อลงแช่ตอนน้ำเต็มอ่าง | ร่างกายช็อคจากการสูญเสียความร้อนอย่างเฉียบพลันในน้ำยะเยือก → ตาย |
| **น้ำล้นอ่าง** | ไม่ปิดน้ำเมื่อเต็มอ่าง ปล่อยให้ไหลล้นออกมา | น้ำไหลท่วมพื้น ไหลไปโดนไดร์เป่าผมที่เสียบปลั๊กอยู่ (ถ้ายังไม่ถอดปลั๊ก) → ไฟดูดตาย / น้ำท่วมพื้นจำนวนมาก → ลื่นล้ม |
| **ไฟดูด (ไดร์ตอนเปียก)** | เดินผ่านหรือจับไดร์เป่าผมที่เสียบปลั๊กอยู่ หลังจากแช่น้ำเสร็จโดยไม่ได้ใช้ผ้าเช็ดตัวให้แห้ง (ตัวเปียก) | ไฟดูดจากไดร์ที่ยังเสียบปลั๊ก + ตัวเปียก → ตาย |

| สถานการณ์ | เงื่อนไข | ผลที่เกิด |
|-----------|----------|----------|
| **ผลข้างเคียง/ยาติดคอ** | เลือกทาน "กระปุกที่ 2 (แคปซูลสีเหลืองสด)" หรือ "กระปุกที่ 5 (เม็ดใหญ่ทรงรีสีส้ม)" | เกิดผลข้างเคียงหรือยาเม็ดใหญ่ติดคอสำลัก ช็อคเบาๆ → ได้รับบาดเจ็บ (ลด HP) |
