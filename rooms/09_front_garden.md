# สวนหน้าบ้าน (Front Garden)

## บรรยากาศ
สวนขนาดกลาง หญ้าสูงผิดปกติสำหรับบ้านที่ดูแลสวนอยู่ ต้นไม้ใหญ่ 3 ต้นในสวน ทางเดินซีเมนต์แตกร้าว ตู้ไปรษณีย์ริมรั้วประตู แสงเช้าที่ดูสว่างแต่ไม่อุ่น — และเสียงรอบข้างเงียบกว่าปกติมาก ไม่มีเสียงนก ไม่มีเสียงรถ

## เงื่อนไขการเข้า
- ต้องมี **กุญแจประตูหน้าบ้าน** (จากห้องซักล้าง หรือกล่องลับห้องนั่งเล่น)

## วัตถุที่โต้ตอบได้

| วัตถุ | การกระทำ | ผล |
|------|----------|-----|
| **ทางเดินซีเมนต์** | เดินตาม | ปลอดภัยถ้าเดินตามแผนที่ |
| **หญ้า / ดิน** | เหยียบ | บางจุดมีกับดัก → **ตาย** |
| **แผนที่สวน (Inventory)** | ใช้ | แสดง Overlay ทับหน้าจอ — ทางเดินปลอดภัยเป็นสีเขียว |
| **ตู้ไปรษณีย์** | เปิด | พบ **จดหมาย** ที่มีตัวเลข → เลขหลักที่ 4 ของรหัสรั้ว |
| **ต้นไม้ใหญ่ (ต้นที่ 1)** | มองขึ้น | มีอะไรแขวนอยู่บนกิ่ง — อย่าใกล้เกินไป |
| **ต้นไม้ใหญ่ (ต้นที่ 2)** | ตรวจสอบโคน | มี **กุญแจสำรอง** ฝังอยู่ในดิน (เผื่อกรณีกุญแจรั้วหาย) |
| **กระถางต้นไม้เล็ก** | พลิกกระถาง | พบ **กระดาษโน้ต** — คำใบ้สุดท้ายเรื่องตัวเอก |
| **รั้วบ้าน** | เดินไปที่รั้ว | เข้าสู่ปริศนาประตูรั้ว |

## ปริศนา

### ปริศนา A — ทางเดินผ่านสวน
- **กฎ:** สวนมีกับดัก 5 จุด กระจายอยู่ในหญ้า
- **วิธีที่ถูก:** ใช้แผนที่สวน (จากใต้โซฟาห้องนั่งเล่น) → เห็น overlay ทางปลอดภัย
- **วิธีที่ผิด:** เดินลงหญ้าโดยไม่มีแผนที่ → เหยียบกับดัก → **ตาย**
- **ทางเลือก:** ใช้ไม้กวาด (จากโถงทางเดิน) เคาะดินทดสอบก่อนเดิน (ช้ากว่าแต่ปลอดภัย)
- **ต้องการ Backtrack:** ต้องมีแผนที่จากห้องนั่งเล่นก่อน

### ปริศนา B — ตู้ไปรษณีย์
- **เนื้อหา:** จดหมายที่ไม่ได้เปิด ส่งถึงชื่อตัวเอก วันที่ประหลาด
- **ตัวเลขในจดหมาย:** เลขบ้านที่เขียนผิด — ตัวเลขที่ผิดนั้นคือ **เลขหลักที่ 4** ของรหัสรั้ว
- **เบาะแสเพิ่มเติม:** เนื้อหาจดหมายเล่าว่าเกิดอะไรขึ้นกับครอบครัวของตัวเอก

## เงื่อนไขการตาย

| การกระทำ | สาเหตุการตาย |
|----------|-------------|
| เหยียบหญ้าโดยไม่มีแผนที่หรือไม้กวาด | กับดักฝังดิน |
| เข้าใกล้ต้นไม้ต้นที่ 1 | สิ่งที่แขวนบนกิ่งตกลงมา |
| อยู่ในสวนนานเกิน 5 นาที (เวลาเกม) | หญ้าขยับเข้าหา — บางอย่างเคลื่อนที่ใต้ดิน |

## สิ่งที่เปลี่ยนเมื่อกลับมา

- ครั้งที่ 2: หญ้าสูงขึ้น กับดักในสวนเพิ่มขึ้น 2 จุด (แผนที่อัปเดตเองอัตโนมัติ)
- ครั้งที่ 3+: ต้นไม้ต้นที่ 3 ที่เคยปกติเริ่มโยกเองในวันที่ไม่มีลม

## Backtracking จากห้องอื่น

| กลับมาจาก | เหตุผล |
|-----------|--------|
| ห้องนั่งเล่น | มาหาแผนที่สวนก่อนเดินในสวน |

## ไอเทมที่ได้จากห้องนี้

- `จดหมาย` → บันทึกเลขหลักที่ 4 ในสมุด
- `กระดาษโน้ตจากกระถาง` → เบาะแสสุดท้ายของเรื่องราว
- `กุญแจสำรอง` (optional) → ไม่ได้ใช้ในปริศนาหลัก
