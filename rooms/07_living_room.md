# ห้องนั่งเล่น (Living Room)

## บรรยากาศ
ห้องกว้างที่สุดในบ้าน โซฟาหนังเก่าตรงกลาง ทีวีจอใหญ่ปิดอยู่ แต่มีแสงสะท้อนกะพริบจากหน้าจอเป็นระยะ ชั้นหนังสือสูงตลอดผนัง หนังสือพิมพ์เก่าหลายฉบับวางซ้อนกัน ประตูหน้าบ้านทางด้านซ้าย — ล็อคอยู่ด้วยกลอนหลายชั้น

## เงื่อนไขการเข้า
- ลงบันไดจากโถงทางเดิน — เข้าได้ทันทีที่ลงบันไดสำเร็จ

## วัตถุที่โต้ตอบได้

| วัตถุ | การกระทำ | ผล |
|------|----------|-----|
| **ทีวี** | เปิด (ต้องถูกเวลา) | แสดงลำดับตัวเลข/สัญลักษณ์ → รหัสกล่องลับในโถง |
| **รีโมทคอนโทรล** | ค้นหา | ซ่อนอยู่ใต้เบาะโซฟา |
| **โซฟา** | ตรวจสอบใต้โซฟา | พบ **แผนที่สวนหน้าบ้าน** + รีโมท |
| **ชั้นหนังสือ** | ค้นหา | พบ **คู่มือเครื่องซักผ้า** + หนังสือเกี่ยวกับบ้าน |
| **หนังสือพิมพ์** | อ่านฉบับที่มีรอยขีด | บันทึก **"เลขหลักที่ 3"** ของรหัสรั้วลงสมุด |
| **กล่องลับบนชั้นหนังสือ** | เปิด (ต้องมีกุญแจจากห้องทานข้าว) | ได้ **กุญแจประตูหน้าบ้าน (สำรอง)** + เบาะแสสุดท้าย |
| **ประตูหน้าบ้าน** | เปิด (ต้องมีกุญแจ) | ออกสู่สวนหน้าบ้าน |
| **นาฬิกาตั้งพื้น** | ตรวจสอบ | บอกเวลาที่ "ถูกต้อง" สำหรับเปิดทีวี |
| **พรมกลางห้อง** | พลิก | พบรอยประทับ — ส่วนหนึ่งของปริศนาเรื่องราว |

## ปริศนา

### ปริศนา A — เปิดทีวีถูกเวลา
- **กฎ:** ทีวีจะแสดงรหัสก็ต่อเมื่อเปิดในช่วงเวลา 6:00–6:05 น. เท่านั้น (เวลาในเกม)
- **วิธีรู้เวลา:** นาฬิกาตั้งพื้นบอกเวลาในเกม
- **รหัสที่ได้:** 3 หลัก สำหรับกล่องลับในโถงทางเดิน
- **ถ้าเปิดทีวีผิดเวลา:** ทีวีแสดงภาพรบกวน + เสียงดัง → ดึงดูด "สิ่งนั้น" → **ตาย**
- **ต้องการ Backtrack:** หลังได้รหัสแล้ว → กลับโถงเพื่อเปิดกล่องลับ

### ปริศนา B — กล่องลับบนชั้นหนังสือ
- **ต้องมีกุญแจ:** จากลิ้นชักใต้เก้าอี้ห้องทานข้าว
- **รางวัล:** กุญแจประตูหน้าบ้าน (ทางเลือกสำรอง) + เอกสารที่เล่าเรื่องราวว่าเกิดอะไรขึ้น

### ปริศนา C — หนังสือพิมพ์ (เบาะแสรหัสรั้ว)
- มีหนังสือพิมพ์หลายฉบับ — ต้องหาฉบับที่มีรอยขีดด้วยปากกาแดง
- วันที่ที่ขีดไว้ = ตัวเลขหลักที่ 3 ของรหัสรั้ว

## เงื่อนไขการตาย

| การกระทำ | สาเหตุการตาย |
|----------|-------------|
| เปิดทีวีผิดเวลา | เสียงดึงดูดสิ่งในบ้าน |
| นั่งโซฟานานเกิน 2 นาทีติดต่อกัน | ตัวเองหลับไม่รู้ตัว — ตื่นอีกทีอยู่ในที่มืด (death screen) |
| พยายามออกประตูหน้าบ้านโดยไม่มีกุญแจ (งัด) | กลอนระเบิดสะเก็ดโดน |

## สิ่งที่เปลี่ยนเมื่อกลับมา

- ครั้งที่ 2: หนังสือบนชั้นบางเล่มหล่นลงพื้น เปิดอยู่ที่หน้าที่มีรูปบ้านนี้
- ครั้งที่ 3+: ทีวีเปิดเองอยู่แล้ว แต่แสดงภาพผู้เล่นในมุมกล้องวงจรปิด

## Backtracking จากห้องอื่น

| กลับมาจาก | เหตุผล |
|-----------|--------|
| โถงทางเดิน | มาดูรหัสจาก TV เพื่อกลับไปเปิดกล่องโถง |
| ห้องซักล้าง | มาหาคู่มือเครื่องซักผ้าก่อนเปิดเครื่อง |

## ไอเทมที่ได้จากห้องนี้

- `รีโมทคอนโทรล` → ใช้เปิดทีวี
- `แผนที่สวน` → ใช้หลีกเลี่ยงกับดักในสวนหน้าบ้าน
- `คู่มือเครื่องซักผ้า` → ใช้ในห้องซักล้าง
- `เลขรหัสหลักที่ 3` (บันทึกในสมุด) → ใช้ที่รั้ว
- `รหัสกล่องโถง` (บันทึกในสมุด) → ใช้ที่โถงทางเดิน
- `กุญแจประตูหน้าบ้าน` (จากกล่องลับ) → เปิดประตูออกสวน
