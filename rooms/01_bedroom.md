# ห้องนอน (Bedroom)

## วัตถุประสงค์
รีบตื่นนอน จัดการความผิดปกติในห้อง และออกจากห้องอย่างปลอดภัย

## บรรยากาศ
ห้องนอนเล็กๆ ชั้นสอง แสงสลัวลอดผ่านหน้าต่างที่เปิดๆ ปิดๆ เป็นจังหวะ นาฬิกาปลุกดังแหลมไม่ยอมหยุด บานหน้าต่างดังลั้นตามลมแรงกระแทก พัดลมเพดานหมุนสลายไปมาแรงขึ้นเรื่อยๆ ประตูตู้เสื้อผ้าปิดไม่สนิทสั่นกระตุก ทุกอย่างดูไม่ปกติ

## สถานะเริ่มต้น
- ห้องแรกที่เข้าได้ — ไม่มีเงื่อนไข
- ตัวละครนอนอยู่บนเตียง ยังไม่ลุก
- **ประตูห้องน้ำ:** ล็อค (จะแง้มเปิดอัตโนมัติเมื่อหยิบผ้าเช็ดตัวที่แขวนอยู่ที่ประตูตู้สำเร็จ)
- **ประตูออกโถง:** ล็อคในตอนแรก (จะต้องไปหากุญแจห้องนอนจากห้องน้ำก่อนเพื่อนำมาไขเปิดออกในภายหลัง)

---

## ความผิดปกติในห้อง (Anomalies)

| ความผิดปกติ | พฤติกรรม | สถานะ |
|------------|----------|-------|
| **นาฬิกาปลุก** | ดังแหลมไม่ยอมหยุด ตั้งอยู่บนโต๊ะข้างเตียง | ต้องหยิบขึ้นมาปิด |
| **กระดาษโน้ต** | วางอยู่ใต้นาฬิกาปลุก (เห็นหลังหยิบนาฬิกาขึ้น) | อ่านเบาะแสสำหรับด่านถัดไป |
| **หน้าต่าง** | เปิดๆ ปิดๆ เอง (แกว่งไปมา) — ต้องจับจังหวะ | ต้องดึงปิดในจังหวะที่บานปิดเข้ามา |
| **พัดลมเพดาน** | หมุนสลายไปมาแรงขึ้นเรื่อยๆ | จะหมุนเบาลงหลังปิดหน้าต่างสำเร็จ |
| **ประตูตู้เสื้อผ้า** | ปิดไม่สนิท สั่นกระตุกเล็กน้อย | จะนิ่งขึ้นหลังปิดหน้าต่าง → ต้องปิดให้สนิท |
| **ผ้าเช็ดตัว** | แขวนอยู่ที่ประตูตู้เสื้อผ้า | หยิบได้หลังปิดประตูตู้สนิทแล้ว |

---

## วัตถุที่โต้ตอบได้

| วัตถุ | การกระทำ | เงื่อนไข | ผล |
|------|----------|----------|-----|
| **เตียง** | ลุกขึ้นนั่ง | — | เริ่มเล่น ตัวละครลุกขึ้นนั่งบนเตียง |
| **นาฬิกาปลุก** | หยิบขึ้นมาปิด | ต้องลุกขึ้นนั่งก่อน | เสียงนาฬิกาหยุด + เผยกระดาษโน้ตที่วางอยู่ข้างใต้ |
| **กระดาษโน้ต** | อ่าน | หยิบนาฬิกาปลุกแล้ว | เบาะแสสำหรับด่านห้องน้ำ: _"อย่าลืมทานยาเม็ดสีชมพูเข้มนะ"_ |
| **หน้าต่าง** | ดึงปิด | ต้องลงจากเตียงแล้ว | **ต้องจับจังหวะ:** ดึงตอนบานปิดเข้ามา = สำเร็จ → ลมหยุด → พัดลมหมุนเบาลง / ตู้เสื้อผ้านิ่งขึ้น |
| **พัดลมเพดาน** | เดินผ่าน (ลอดใต้) | พัดลมต้องหมุนเบาลงแล้ว (หลังปิดหน้าต่าง) | สามารถเดินผ่านไปยังฝั่งตู้เสื้อผ้าได้ |
| **ประตูตู้เสื้อผ้า** | ปิดให้สนิท | ประตูตู้ต้องนิ่งขึ้นแล้ว (หลังปิดหน้าต่าง) | ปิดสำเร็จ → สามารถหยิบผ้าเช็ดตัวได้ |
| **ผ้าเช็ดตัว** | หยิบ | ปิดประตูตู้เสื้อผ้าแล้ว | ประตูห้องน้ำแง้มเปิดอัตโนมัติ (ปลดล็อคห้องน้ำ) |
| **ประตูห้องน้ำ** | เปิด | หยิบผ้าเช็ดตัวแล้ว | ✅ เข้าห้องน้ำได้ → ไปยังด่านถัดไป |
| **ประตูออกโถง** | พยายามเปิด | — | ล็อคอยู่ เปิดไม่ได้ |

## Win Flow (ลำดับการเล่นที่ถูกต้อง)

```
① ลุกขึ้นนั่งบนเตียง
    │
② หยิบนาฬิกาปลุกขึ้นมาปิด
    │
③ พบกระดาษโน้ต (เบาะแสด่านถัดไป)
    │
④ ลงจากเตียง → เดินไปที่หน้าต่าง
    │
⑤ จับจังหวะดึงหน้าต่างปิด ← ดึงตอนบานปิดเข้ามา
    │
    ├─→ พัดลมเพดานหมุนเบาลง
    └─→ ประตูตู้เสื้อผ้านิ่งขึ้น
    │
⑥ รอจังหวะ → เดินลอดผ่านพัดลมเพดาน
    │
⑦ ปิดประตูตู้เสื้อผ้าให้สนิท
    │
⑧ หยิบผ้าเช็ดตัวที่แขวนอยู่ที่ประตูตู้
    │
⑨ ประตูห้องน้ำแง้มเปิดอัตโนมัติ (🔓 ปลดล็อค)
    │
⑩ เดินไปเข้าห้องน้ำ → 🏁 ผ่านด่าน
```

## เบาะแส

| แหล่ง | เนื้อหา |
|-------|---------|
| **กระดาษโน้ต** (ใต้นาฬิกาปลุก) | _"อย่าลืมทานยาเม็ดสีชมพูเข้มนะ"_ → ใบ้สำหรับด่านห้องน้ำ |
| **ประตูออกโถง** | ล็อคอยู่ → บังคับให้ผู้เล่นไปทางห้องน้ำแทน |

## เงื่อนไขการตาย ☠️

| สถานการณ์ | เงื่อนไข | สาเหตุการตาย |
|-----------|----------|-------------|
| **ตกตึก** | ดึงปิดหน้าต่างในจังหวะที่บานอ้าออกข้างนอก | ตัวละครโน้มตัวตามแรง ถูกดึงออกไปนอกหน้าต่าง ตกจากชั้นสอง |
| **ใบพัดหลุด** | เข้าใกล้พัดลมเพดานตอนที่ยังหมุนแรงอยู่ (ก่อนปิดหน้าต่าง) | พัดลมส่ายแรงจนใบพัดหลุดออกมา บาดร่างกาย หรือหล่นทับ |
| **ไม่ปิดหน้าต่าง** | ปล่อยไว้นานเกินไปโดยไม่ปิดหน้าต่าง | พัดลมหมุนแรงขึ้นจนใบพัดหลุด — ตายเหมือนกรณีข้างบน |
| **ประตูตู้ทับ** | ดึงผ้าเช็ดตัวที่ประตูตู้เสื้อผ้าตอนที่ประตูยังสั่นอยู่ (ก่อนปิดหน้าต่าง) | ประตูตู้หลุดจากบานพับ ล้มทับตัวละคร |

## เงื่อนไขการบาดเจ็บ 🩹

> [!WARNING]
> การบาดเจ็บสะสมหลายครั้งจะทำให้ตายได้เช่นกัน

| สถานการณ์ | เงื่อนไข | ผลที่เกิด |
|-----------|----------|----------|
| **นาฬิการ่วงใส่หน้า** | เอื้อมหยิบนาฬิกาปลุกก่อนลุกขึ้นนั่ง (ยังนอนอยู่) | นาฬิการ่วงใส่หน้า เพราะยังงัวเงีย — ได้รับบาดเจ็บ |
| **แตะขอบเตียง** | เดินออกจากเตียงไปจุดใกล้ๆ ก่อนปิดนาฬิกาปลุก | เดินสะดุดแตะขอบเตียง เพราะยังงัวเงีย — ได้รับบาดเจ็บ |
| **โดนหนีบมือ** | พยายามปิดประตูตู้เสื้อผ้าตอนที่ยังสั่นแรง (ก่อนปิดหน้าต่าง) | ประตูตู้สั่นหนีบมือ — ได้รับบาดเจ็บ |
| **กลิ้งตกเตียง** | เดินออกจากเตียงแล้วเลือกไปที่ประตูเลย (ไม่ทำอะไรก่อน) | ตัวละครรีบร้อนเกิน กลิ้งตกจากเตียง — ได้รับบาดเจ็บ |

## ไอเทมที่ได้จากห้องนี้

- `ผ้าเช็ดตัว` → ทริกเกอร์ปลดล็อคห้องน้ำ (ไม่ได้เก็บเข้า Inventory)

## ทางออกจากห้อง

| ทางออก | สถานะ | เงื่อนไข |
|--------|-------|----------|
| **ประตูห้องน้ำ** | 🔒→🔓 | หยิบผ้าเช็ดตัวแล้ว ประตูจะแง้มเปิดอัตโนมัติ |
| **ประตูออกโถง** | 🔒→🔓 | ล็อคตอนเริ่มเกม เมื่อได้กุญแจห้องนอนจากอ่างอาบน้ำแล้ว ให้เดินย้อนมาไขเพื่อเปิดออกสู่โถง |
