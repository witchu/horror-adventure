# ห้องทานข้าว (Dining Room)

## วัตถุประสงค์
ดื่มชามิ้นต์ ค้นหากุญแจห้องเก็บของที่ซ่อนอยู่บนโคมไฟเพดาน และซ่อมล้อนาฬิกาลูกตุ้มเพื่อเลื่อนออกจากประตูทางไปสู่ห้องนั่งเล่น

## เงื่อนไขการเข้า
- เข้ามาจากห้องครัวผ่านทางเชื่อมระหว่างสองห้อง หรือกลับมาจากห้องนั่งเล่น
- **เงื่อนไขพิเศษ:** ชุดเครื่องดื่มบนโต๊ะ (กาชามิ้นท์, กาต้มกาแฟ, เหยือกน้ำ, แก้วเปล่า) จะ**ไม่ปรากฏ**บนโต๊ะทานข้าว หากผู้เล่นยังไม่ได้ชิมอาหารที่ปรุงรสแล้วจากห้องครัว เมื่อชิมอาหารสำเร็จแล้ว จะมีเสียงดังกุกกักจากห้องทานข้าวทันที และชุดเครื่องดื่มจะปรากฏขึ้นบนโต๊ะก่อนที่ผู้เล่นจะเดินมาถึง

## บรรยากาศ
ห้องทานข้าวสว่างสลัวด้วยแสงโคมไฟเพดานที่ติดๆ ดับๆ เป็นจังหวะ ทำให้เงาในห้องหยักย้ายเต้นอยู่ตลอดเวลา กลิ่นชาอุ่นๆ ล่องลอยออกมาจากกาต้มที่ตั้งรออยู่บนโต๊ะไม้เก่า เสียงเคาะสม่ำเสมอของนาฬิกาลูกตุ้มที่ตั้งขวางประตูห้องนั่งเล่นสร้างจังหวะที่ทั้งนิ่งและน่าอึดอัด หนังสือพิมพ์พับเรียบร้อยวางรออยู่ราวก็บอกเป็นนัยว่านี่คือเช้าวันปกติ — แต่ดูเหมือนจะไม่มีทางออกจากที่นี่ไปได้ง่ายๆ

## สถานะเริ่มต้น
- โคมไฟเพดานกลางโต๊ะทานข้าวกำลังติดๆ ดับๆ เป็นจังหวะ
- สวิตซ์ไฟอยู่ในสถานะเปิด (ON)
- นาฬิกาลูกตุ้มตั้งขวางประตูห้องนั่งเล่น เคาะดังเป็นจังหวะ แต่ล้อเลื่อนพัง ไม่สามารถขยับตัวนาฬิกาออกได้
- หนังสือพิมพ์พับเรียบร้อยวางอยู่บนโต๊ะ
- **ชุดเครื่องดื่มยังไม่ปรากฏ** จนกว่าจะชิมอาหารที่ปรุงรสแล้วจากห้องครัว
- ประตูห้องนั่งเล่นถูกล็อคโดยนาฬิกาลูกตุ้มที่ตั้งขวางทางอยู่

## ความผิดปกติในห้อง (Anomalies)

| ความผิดปกติ | พฤติกรรม | สถานะ |
|-------------|----------|-------|
| โคมไฟเพดาน | กระพริบติดๆ ดับๆ เป็นจังหวะไม่สม่ำเสมอ สร้างแสงสว่างทั่วห้อง | แก้ไขได้โดยปิด-เปิดสวิตซ์ไฟใหม่ให้ไฟสว่างเต็มที่ |
| แสงวิ้งวับบนโคมไฟ | ขณะไฟกระพริบในจังหวะดับมืด หรือ ปิดสวิตซ์ให้ไฟดับสนิท จะมองเห็นแสงวิ้งวับที่ขอบโคมไฟ — กุญแจห้องเก็บของซ่อนอยู่ | กุญแจปรากฏชัดขึ้นในความมืด แต่ต้องเข้าถึงให้ถูกจังหวะ |
| นาฬิกาลูกตุ้ม | เคาะดังเป็นจังหวะ ตั้งขวางประตูห้องนั่งเล่น ล้อเลื่อนด้านล่างพัง ขยับไม่ได้ | ต้องซ่อมล้อก่อนถึงจะเลื่อนออกจากประตูได้ |

## วัตถุที่โต้ตอบได้

| วัตถุ | การกระทำ | เงื่อนไข | ผล |
|------|----------|----------|-----|
| **สวิตซ์ไฟ** | ปิด (ครั้งที่ 1) | สวิตซ์กำลังเปิดอยู่ (ไฟกระพริบ) | ไฟดับสนิท — มองเห็นแสงวิ้งวับบนโคมไฟชัดเจน |
| **สวิตซ์ไฟ** | เปิด (ครั้งที่ 2) | สวิตซ์ถูกปิดไปแล้ว (ไฟดับ) | ไฟกลับมาสว่างเต็มที่ชั่วขณะหนึ่ง ใช้โอกาสนี้ดื่มและปีนโต๊ะได้ปลอดภัย |
| **สวิตซ์ไฟ** | ปิด (ครั้งที่ 3) | หลังดื่มชามิ้นต์แล้ว | ไฟดับสนิทอีกครั้ง — เพื่อเตรียมปีนโต๊ะหยิบกุญแจ |
| **กาชามิ้นท์** | ดื่มชามิ้นท์ | ชุดเครื่องดื่มปรากฏบนโต๊ะ และไฟสว่างเต็มที่อยู่ | ผู้เล่นสงบสติอารมณ์ ลด panic ทำให้ปีนโต๊ะได้อย่างมั่นคง |
| **กาต้มกาแฟ** | ดื่มกาแฟ | ชุดเครื่องดื่มปรากฏบนโต๊ะ | ทำให้ระบบประสาทกระตุ้นมากเกินไป panic กำเริบ (**ทางตายที่มีทางแก้**) |
| **เหยือกน้ำเปล่า** | ดื่มน้ำเปล่า | ชุดเครื่องดื่มปรากฏบนโต๊ะ | ช่วยระงับ panic จากกาแฟได้ หากดื่มก่อนนาฬิกาดังครบ 5 ครั้ง |
| **แก้วเปล่า** | หยิบ / วางเครื่องดื่มใส่แก้ว | ชุดเครื่องดื่มปรากฏบนโต๊ะ | ใช้เป็นภาชนะรับเครื่องดื่ม |
| **หนังสือพิมพ์** | อ่าน / กางออก | — | พบวันที่ที่ถูกขีดเส้นใต้ด้วยปากกา — บันทึกเบาะแสรหัสรั้วหลักที่ 2 |
| **โต๊ะทานข้าว** | ปีนขึ้นไป | ต้องดื่มชามิ้นต์ก่อน + ไฟต้องดับสนิท | เข้าถึงโคมไฟเพดานได้ โดยไม่สั่นกลัวจาก panic |
| **โคมไฟเพดาน** | หยิบกุญแจ | ปีนโต๊ะสำเร็จ + ไฟต้องดับสนิทอยู่ (ห้ามเปิดแฟลชสมาร์ทโฟน) | ได้รับ **กุญแจห้องเก็บของ** ที่ซ่อนอยู่บนขอบโคมไฟ |
| **นาฬิกาลูกตุ้ม** | ตรวจสอบล้อเลื่อน | — | พบว่าล้อพัง ต้องการ **ชุดอุปกรณ์ซ่อมล้อ** จากห้องซักล้าง |
| **นาฬิกาลูกตุ้ม** | เลื่อนนาฬิกาออก | มี **ชุดอุปกรณ์ซ่อมล้อ** ในInventory + ซ่อมล้อแล้ว | นาฬิกาเลื่อนออกจากทางประตู เปิดทางสู่ห้องนั่งเล่น |
| **ประตูห้องนั่งเล่น** | เปิดประตู | นาฬิกาถูกเลื่อนออกแล้ว | เข้าสู่ห้องนั่งเล่นได้ |
| **ประตูห้องครัว** | เดินกลับ | — | กลับไปห้องครัวได้ตลอดเวลา |

## Win Flow (ลำดับการเล่นที่ถูกต้อง)

```text
[เงื่อนไขก่อนเข้า] ชิมอาหารที่ปรุงรสแล้วจากห้องครัว
    │
    └─→ เสียงดังกุกกักจากห้องทานข้าว — ชุดเครื่องดื่มปรากฏบนโต๊ะ
    │
① ปิด-เปิดสวิตซ์ไฟ (ปิดแล้วเปิดใหม่) เพื่อให้ไฟสว่างเต็มที่
    │
② ดื่มชามิ้นต์ (ขณะไฟสว่างเต็มที่)
    │
③ ปิดสวิตซ์ไฟให้ดับสนิท
    │
④ ปีนโต๊ะทานข้าว (ทำได้ปลอดภัยเพราะดื่มชามิ้นต์แล้ว)
    │
⑤ หยิบกุญแจห้องเก็บของจากโคมไฟ (ขณะไฟดับ — กุญแจสะท้อนแสงวิ้งวับ)
    │
    ├─→ [แวะ] อ่านหนังสือพิมพ์ → บันทึกเบาะแสรหัสรั้วหลักที่ 2 (ทำหรือไม่ก็ได้)
    │
⑥ → ออกไปห้องเก็บของ (ใช้กุญแจที่ได้)
    │
    └─→ → ผ่านห้องเก็บของ → ห้องซักล้าง → รับชุดอุปกรณ์ซ่อมล้อ
    │
⑦ กลับมาห้องทานข้าว — ซ่อมล้อนาฬิกาลูกตุ้ม
    │
⑧ เลื่อนนาฬิกาลูกตุ้มที่ซ่อมแล้วออกจากทางประตู
    │
    └─→ เปิดประตูสู่ห้องนั่งเล่น ✓
```

## เบาะแส

| แหล่ง | เนื้อหา |
|-------|---------|
| หนังสือพิมพ์บนโต๊ะ | วันที่ในหนังสือพิมพ์ที่ถูกขีดเส้นใต้ไว้ = **รหัสรั้วหลักที่ 2** |
| โคมไฟเพดาน (ขณะไฟดับสนิท) | แสงวิ้งวับที่ขอบโคมไฟ บ่งชี้ว่ามีสิ่งของซ่อนอยู่บนนั้น (การเปิดแฟลชส่องจะไม่ช่วยให้โคมไฟสว่างขึ้น แสงจ้าจากสมาร์ทโฟนจะกลืนประกายแสงนี้ ทำให้ผู้เล่นถูกบังคับให้ต้องปิดไฟห้องเพื่อหาในความมืดเท่านั้น) |
| นาฬิกาลูกตุ้ม | ล้อเลื่อนพัง ต้องการของมาซ่อมก่อนถึงจะขยับได้ — ต้องหาจากห้องซักล้าง |

## เงื่อนไขการตาย ☠️

| สถานการณ์ | เงื่อนไข | สาเหตุการตาย |
|-------------|----------|-------------|
| นาฬิกาล้มทับ | พยายามเลื่อนนาฬิกาลูกตุ้มที่ยังไม่ได้ซ่อมล้อออกจากประตู | ล้อที่พังทำให้นาฬิกาเอียงและล้มทับตัวผู้เล่น |
| ตกจากโต๊ะ — หัวฟาดพื้น | ปีนโต๊ะทานข้าวก่อนที่จะดื่มชามิ้นต์ | อาการ panic กำเริบระหว่างปีน ตัวสั่น เสียการทรงตัวและตกลงมาหัวกระแทกพื้นอย่างรุนแรง |
| ไฟดูดบนโคมไฟ | หยิบกุญแจบนโคมไฟเพดานขณะที่ไฟยังกระพริบอยู่ หรือขณะที่เปิดไฟสว่างเต็มที่ | กระแสไฟฟ้าจากโคมไฟที่ผิดปกติวิ่งเข้าร่างกาย |
| ช็อกตาย — panic กำเริบรุนแรง | ดื่มกาแฟ และได้ยินเสียงนาฬิกาดังครบ 5 ครั้งโดยไม่ดื่มน้ำเปล่าแก้กันทัน | ระบบประสาทที่ถูกกระตุ้นจากคาเฟอีน + เสียงดัง ทำให้ panic กำเริบรุนแรงจนหัวใจวาย |

> [!NOTE]
> **ทางแก้กรณีดื่มกาแฟ:** หากผู้เล่นดื่มกาแฟ จะได้ยินเสียงนาฬิกาเริ่มนับ — ต้องรีบดื่มน้ำเปล่าจากเหยือกให้ทันก่อนนาฬิกาดัง**ครบ 5 ครั้ง** เพื่อระงับ panic และหลีกเลี่ยงการตาย

## เงื่อนไขการบาดเจ็บ 🩹

> [!WARNING]
> การบาดเจ็บสะสมอาจทำให้พลังชีวิต (HP) หมดและ Game Over ได้หากไม่ระวัง

| สถานการณ์ | เงื่อนไข | ผลที่เกิด |
|-------------|----------|----------|
| ถูกน้ำร้อนลวก | ดื่มเครื่องดื่มใดๆ (ชามิ้นต์ / กาแฟ / น้ำเปล่า) ขณะที่ไฟยังกระพริบหรือดับสนิท | มือสั่นเพราะมองไม่ถนัด ทำน้ำร้อนหก เผลอถูกน้ำร้อนลวกมือหรือลำตัว ลด HP |

## สิ่งที่เปลี่ยนเมื่อกลับมา
- โคมไฟเพดานหยุดกระพริบ (ไฟดับสนิทหลังหยิบกุญแจ หากผู้เล่นไม่ได้เปิดสวิตซ์ใหม่)
- ชุดเครื่องดื่มบนโต๊ะยังคงอยู่ แต่ไม่มีเหตุการณ์ใหม่จากการดื่มอีก
- นาฬิกาลูกตุ้มอยู่ในสถานะรอซ่อม (หากยังไม่ได้ซ่อม) หรืออยู่ข้างผนังแล้ว (หากเลื่อนออกไปแล้ว)
- ประตูห้องนั่งเล่นเปิดได้เสรีหลังจากเลื่อนนาฬิกาออกสำเร็จ
- บรรยากาศห้องเงียบขึ้นอย่างมาก ไม่มีเสียงเคาะนาฬิกา

## Backtracking จากห้องอื่น
- ผู้เล่นต้องกลับมาห้องนี้หลังจากผ่านห้องเก็บของ → ห้องซักล้าง เพื่อนำ **ชุดอุปกรณ์ซ่อมล้อ** มาซ่อมนาฬิกาลูกตุ้มและเลื่อนออกจากประตูห้องนั่งเล่น

## ไอเทมที่ได้จากห้องนี้
- **กุญแจห้องเก็บของ** — เก็บเข้า Inventory (หยิบจากโคมไฟเพดานขณะปีนโต๊ะในความมืด)
- **เบาะแสรหัสรั้วหลักที่ 2** — บันทึกอัตโนมัติในสมุดบันทึก (จากวันที่ที่ขีดไว้ในหนังสือพิมพ์)
