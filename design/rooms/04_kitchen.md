# ห้องครัว (Kitchen)

## วัตถุประสงค์
ปรุงอาหารให้เสร็จ และจัดการอุปกรณ์ต่างๆ ในครัวให้เรียบร้อยเพื่อความปลอดภัยก่อนออกจากห้อง

## เงื่อนไขการเข้า
- เข้ามาจากโถงทางเดินชั้น 1 หรือเดินกลับมาจากห้องทานข้าว

## บรรยากาศ
บรรยากาศในครัวดูอบอุ่นแต่อึดอัดและเร่งรีบ มีเสียงน้ำไหลจากก๊อกเบาๆ ผสมกับเสียงหวีดร้องของกาต้มน้ำที่กำลังเดือด กลิ่นอาหารที่เปิดอุ่นทิ้งไว้บนเตาเริ่มส่งกลิ่นเหม็นไหม้ ทุกอย่างดูพร้อมจะก่อให้เกิดอันตรายได้ทุกเมื่อหากปล่อยทิ้งไว้

## สถานะเริ่มต้น
- ผู้เล่นเข้ามาในห้องครัวที่เต็มไปด้วยความวุ่นวาย
- น้ำในอ่างล้างจานถูกเปิดไหลทิ้งไว้ 
- กาต้มน้ำกำลังเดือดและเริ่มส่งเสียงหวีดร้อง
- ตู้เก็บจานแขวนผนังปิดไม่สนิท
- เตาแก๊สเปิดอุ่นอาหารทิ้งไว้ และค่อยๆเริ่มมีควัน
- เหตุการณ์ความผิดปกติในห้องครัวจะดำเนินการไปเรื่อยๆ ตามเวลา (Real-time) แม้ผู้เล่นจะเดินหนีไปยังห้องทานข้าวหรือห้องอื่นระหว่างนั้นก็ตาม

## ความผิดปกติในห้อง (Anomalies)
| ความผิดปกติ | พฤติกรรม | สถานะ |
|-------------|----------|-------|
| ก๊อกน้ำอ่างล้างจาน | น้ำเปิดไหลเบาๆ และค่อยๆ สะสมในอ่างล้างจาน | ต้องรีบปิดก๊อกน้ำก่อนที่น้ำจะล้นอ่างลงมาท่วมพื้น |
| กาต้มน้ำ | เดือดจัดและส่งเสียงหวีดร้องดังขึ้นเรื่อยๆ | ต้องรีบปิดกาต้มน้ำก่อนที่แรงดันจะทำให้ระเบิด |
| ตู้เก็บจานแขวนผนัง | ปิดไม่สนิท และบานประตูค่อยๆ เปิดกว้างขึ้นเรื่อยๆ | ต้องรีบผลักปิดบานตู้ให้สนิทก่อนที่จานจะร่วงลงมา |
| อาหารบนเตาแก๊ส | เปิดอุ่นทิ้งไว้นานจนเริ่มมีควันและกลิ่นไหม้ฟุ้งกระจาย | ต้องหาวิธีปิดแก๊สให้สำเร็จเพื่อหยุดควันไหม้ |

## วัตถุที่โต้ตอบได้
| วัตถุ | การกระทำ | เงื่อนไข | ผล |
|------|----------|----------|----|
| **ก๊อกน้ำอ่างล้างจาน** | ปิดน้ำ | - | น้ำหยุดไหล ป้องกันน้ำท่วมพื้นห้อง |
| **กาต้มน้ำ** | ปิดกาต้มน้ำ | - | กาต้มน้ำหยุดเดือดและเสียงเงียบลง ป้องกันการระเบิด |
| **ตู้เก็บจานแขวนผนัง** | ปิดตู้ | - | บานตู้ปิดสนิท ป้องกันจานร่วงหล่นใส่ |
| **ลิ้นชักซ้าย** | เปิดลิ้นชัก | - | พบ **สมุดโน๊ต** เขียนวิธีปิดเตาแก๊ส |
| **ลิ้นชักขวา** | เปิดลิ้นชัก | - | ถูกของมีคมด้านในบาดมือ (บาดเจ็บ) |
| **เตาแก๊ส** | ปิดเตา (หมุน ขวา-ซ้าย-ซ้าย-ขวา) | ต้องอ่านสมุดโน๊ตเพื่อรู้วิธีก่อน | เตาแก๊สถูกปิด อาหารบนเตาหยุดเดือดปุดๆ ควันและกลิ่นไหม้ค่อยๆ จางหายไป |
| **อาหารบนเตา** | ชิมอาหาร (ครั้งแรก) | ต้องปิดเตาแก๊สก่อน (ถ้ายังไม่ปิด อาหารจะกำลังเดือดและร้อนเกินไปจนไม่สามารถชิมได้) | รับรู้ว่ารสชาติอาหารยังขาดไป ต้องหาเครื่องปรุงเพิ่ม |
| **ชั้นวางเครื่องปรุง** | ตรวจสอบ | ต้องชิมอาหารบนเตาไปแล้วหนึ่งรอบ | พบกระปุกเครื่องปรุง 6 กระปุก (กระปุกที่ 1 ถึง 6) เรียงกันอยู่ |
| **กระปุกที่ 1** | สังเกตลักษณะ / นำมาปรุงอาหาร | ตรวจสอบชั้นวางเครื่องปรุงแล้ว | มีลักษณะ: **เกล็ดเล็กๆ สีน้ำตาลอ่อน กลิ่นหอมหวาน** → ปรุงรสปลอดภัย |
| **กระปุกที่ 2** | สังเกตลักษณะ / นำมาปรุงอาหาร | ตรวจสอบชั้นวางเครื่องปรุงแล้ว | มีลักษณะ: **เกล็ดเหลี่ยมขนาดใหญ่กว่าผงอื่น สีขาวขุ่น ไม่มีกลิ่น** → ปรุงรสปลอดภัย |
| **กระปุกที่ 3** | สังเกตลักษณะ / นำมาปรุงอาหาร | ตรวจสอบชั้นวางเครื่องปรุงแล้ว | มีลักษณะ: **ผงละเอียด สีดำ กลิ่นเคมี** → อาหารปนเปื้อน (ยาพิษ) |
| **กระปุกที่ 4** | สังเกตลักษณะ / นำมาปรุงอาหาร | ตรวจสอบชั้นวางเครื่องปรุงแล้ว | มีลักษณะ: **เกล็ดละเอียด สีขาวใส ไม่มีกลิ่น** → ปรุงรสปลอดภัย |
| **กระปุกที่ 5** | สังเกตลักษณะ / นำมาปรุงอาหาร | ตรวจสอบชั้นวางเครื่องปรุงแล้ว | มีลักษณะ: **ผงหยาบ สีน้ำตาลเข้ม/ดำ กลิ่นฉุน** → ปรุงรสปลอดภัย |
| **กระปุกที่ 6** | สังเกตลักษณะ / นำมาปรุงอาหาร | ตรวจสอบชั้นวางเครื่องปรุงแล้ว | มีลักษณะ: **ผงหยาบ สีส้มเข้ม/แดง กลิ่นเผ็ดร้อน** → ปรุงรสปลอดภัย |
| **อาหารที่ปรุงแล้วบนเตา** | ชิมอาหาร (ครั้งที่สอง) | ปรุงอาหารด้วยกระปุกใดกระปุกหนึ่งเสร็จแล้ว | ถ้ารสชาติปลอดภัย = ผ่านเงื่อนไข / ถ้าปนเปื้อนกระปุกที่ 3 = ตาย |
| **กระดานโน๊ตบนตู้เย็น** | อ่านโน๊ต | - | ได้รับเบาะแสเป้าหมายหลักของห้องครัว |
| **ประตูห้องซักล้าง** | ตรวจสอบ | - | ลูกบิดฝืดแน่นด้วยสนิม ไม่สามารถเปิดออกได้ในตอนนี้ |

## Win Flow (ลำดับการเล่นที่ถูกต้อง)
```text

(ขั้นตอน ①, ②, ③, ④, ⑤ สามารถทำอันไหนก่อนก็ได้ แต่ต้องปิดให้ครบและทันเวลาก่อนเกิดอันตราย)
① ปิดก๊อกน้ำอ่างล้างจาน
    │
② ปิดกาต้มน้ำ
    │
③ ปิดตู้เก็บจานแขวนผนังให้สนิท
    │
④ ค้นลิ้นชักฝั่งซ้าย
    │
    └─→ พบสมุดโน๊ตบอกวิธีปิดเตาแก๊ส
    │
⑤ ปิดเตาแก๊ส (หมุน ขวา-ซ้าย-ซ้าย-ขวา ตามสมุดโน๊ต)
    │
⑥ ชิมอาหารบนเตา (ครั้งแรก)
    │
⑦ ตรวจสอบชั้นวาง และเลือกกระปุกเครื่องปรุง (ต้องสังเกตลักษณะและหลีกเลี่ยงกระปุกที่มีสีดำ กลิ่นเคมี)
    │
⑧ นำเครื่องปรุงที่ปลอดภัย (เช่น กระปุกที่ 1, 2, 4, 5 หรือ 6) มาปรุงลงในอาหาร
    │
⑨ ชิมอาหารที่ปรุงแล้วอีกครั้ง
    │
    └─→ อาหารปรุงสำเร็จ เคลียร์ปริศนาและความปลอดภัยในห้องครัว

```

## เบาะแส
| แหล่ง | เนื้อหา |
|-------|---------|
| กระดานโน๊ตบนตู้เย็น | "ทานอาหารด้วยนะ ฉันอุ่นเตรียมไว้ให้แล้ว...แต่รสชาติอาจไม่ถูกใจคุณเท่าไหร่ และอย่าลืมตรวจสอบทุกอย่างให้เรียบร้อยก่อนออกไปด้วยล่ะ" (คำใบ้เป้าหมายของห้องนี้) |
| สมุดโน๊ตในลิ้นชักซ้าย | การปิดเตาแก๊สรุ่นนี้ต้องหมุนวาล์วตามลำดับ: ขวา - ซ้าย - ซ้าย - ขวา |

## เงื่อนไขการตาย ☠️
| สถานการณ์ | เงื่อนไข | สาเหตุการตาย |
|-------------|----------|-------------|
| ลื่นล้มหัวฟาดพื้น | ปล่อยให้น้ำในอ่างล้างจานไหลล้นลงมาท่วมพื้นจนเจิ่งนอง แล้วเดินผ่านบริเวณนั้น | ลื่นน้ำที่ท่วมพื้นล้มหัวฟาดอย่างรุนแรง |
| ถูกกาต้มน้ำระเบิดใส่ | ปิดกาต้มน้ำไม่ทันตามเวลาที่กำหนด หรือปล่อยทิ้งไว้จนแรงดันเกินพิกัด | กาต้มน้ำระเบิดใส่อย่างรุนแรง |
| จานหล่นใส่ | ปิดตู้เก็บจานไม่ทันเวลา จนตู้เปิดกว้างขึ้นเรื่อยๆ แล้วเดินผ่านหรือยืนอยู่ตรงนั้นพอดี | จานจำนวนมากร่วงตกลงมาใส่ศีรษะ |
| โดนยาพิษ | เลือกหยิบผงปรุงผิดจากชั้นวาง (หยิบกระปุกที่ 3 - ผงละเอียด สีดำ กลิ่นเคมี) มาปรุงอาหาร และตักชิมเข้าปาก | พิษเคมีทำลายระบบร่างกายอย่างรุนแรง นำไปสู่ความตาย |

## เงื่อนไขการบาดเจ็บ 🩹
> [!WARNING]
> การบาดเจ็บสะสมอาจทำให้พลังชีวิต (HP) หมดและ Game Over ได้หากไม่ระวัง

| สถานการณ์ | เงื่อนไข | ผลที่เกิด |
|-------------|----------|----------|
| ของมีคมบาด | เปิดลิ้นชักฝั่งขวาอย่างไม่ระมัดระวัง | ถูกของมีคมที่ซ่อนอยู่บาดมือ |
| สูดดมควันไหม้ | ปล่อยอาหารบนเตาอุ่นทิ้งไว้นานๆ โดยไม่รีบหาวิธีปิดแก๊ส | สูดควันจากอาหารไหม้เข้าปอด (ลด HP ต่อเนื่อง จนกว่าจะปิดเตาแก๊สสำเร็จ) |

## สิ่งที่เปลี่ยนเมื่อกลับมา
- สภาพในครัวปลอดภัยและเงียบสงบขึ้น กลิ่นควันจางหายไปแล้ว และไม่มีเสียงหวีดร้องของกาต้มน้ำ 

## Backtracking จากห้องอื่น
- ผู้เล่นจะต้องเดินย้อนกลับมาที่ห้องครัวอีกครั้ง เพื่อเป็นเส้นทางผ่านในการนำค้อนที่ได้มาไปทุบประตูลูกบิดที่ขึ้นสนิม (ประตูห้องซักล้าง) หลังจากได้ค้อนมาจากห้องเก็บของแล้ว

## ไอเทมที่ได้จากห้องนี้
- **ไม่มีไอเทมเก็บเข้า Inventory โดยตรง:** แต่การทำปริศนาปรุงอาหารและชิมสำเร็จ ถือเป็น **Key Trigger หลัก** (ตัวแปรสถานะเกม) ที่จะทำให้เกมเสก "ชุดเครื่องดื่มพร้อมดื่ม" ให้ปรากฏขึ้นในด่านถัดไป (ห้องทานข้าว) อย่างอัตโนมัติ 

## ทางออกจากห้อง
| ทางออก | สถานะ | เงื่อนไข |
|--------|-------|----------|
| โถงทางเดิน | `เปิดอยู่` | เดินกลับสู่โถงทางเดินชั้น 1 ได้ตลอดเวลา |
| ห้องทานข้าว | `เปิดอยู่` | เดินเข้าไปได้ตลอดเวลา แม้ว่าปริศนาในครัวจะยังไม่เสร็จสิ้น (แต่ความผิดปกติในครัวจะยังคงดำเนินต่อไป) |
| ประตูห้องซักล้าง | `🔒` | ลูกบิดแน่นด้วยสนิม เปิดไม่ออก (ต้องหาวิธีคลายสนิมในภายหลัง) |
