# ถนน (Road)

## วัตถุประสงค์
ข้ามถนนไปให้ถึงฝั่งตรงข้ามให้สำเร็จ — นี่คือฉากสุดท้ายก่อนออกไปใช้ชีวิตตามปกติ ผู้เล่นต้องสังเกตสัญญาณไฟจราจรและเลือกจังหวะข้ามให้ถูกต้อง รวมถึงตัดสินใจโต้ตอบกับคนแปลกหน้าในฉากตามที่ต้องการ

## เงื่อนไขการเข้า
- เข้ามาจากรั้วหน้าบ้าน หลังจากกรอกรหัส 4 หลักที่แผงประตูรั้วได้สำเร็จ (0-2-1-0)
- ผู้เล่นเริ่มยืนอยู่บนฟุตบาทฝั่งบ้าน — เห็นถนนที่เปียกชื้น ทางม้าลายที่ไม่มีแสงส่อง สัญญาณไฟจราจรไกลๆ ชายสูบบุหรี่ที่ยืนอยู่ใกล้รั้ว และผู้หญิงยืนใต้เงามืดฝั่งตรงข้าม
- **ฉากนี้ไม่มีทางย้อนกลับ** — เมื่อตัวละครเดินออกจากฉากสู่ฝั่งตรงข้ามสำเร็จ เกมจะจบลง

## บรรยากาศ
ตี 4 ท้องฟ้ายังมืดสนิท อากาศชื้นและหนักหน่วงจากพายุที่เพิ่งผ่านพ้น ถนนลาดยางสีดำมันวาวจากความชื้น สะท้อนแสงไฟจราจรไกลๆ และไฟเสาที่กะพริบๆ พร้อมละอองฝนที่ยังลอยอยู่ในอากาศ ทางม้าลายถัดจากรั้วบ้านออกไปไม่ไกลอยู่ในมุมมืดที่ไฟส่องไม่ถึง สัญญาณไฟจราจรไกลออกไปเปลี่ยนสีเงียบๆ โดยไม่มีใครสนใจ ผู้หญิงคนหนึ่งยืนนิ่งอยู่ฝั่งตรงข้ามใต้เงาของไฟกะพริบ ชายสูบบุหรี่ยืนพิงรั้วบ้านอยู่บนฟุตบาทเดียวกับบ้านของเรา ควันบุหรี่ลอยขึ้นปนหายไปกับละอองฝน ทุกอย่างดูปกติหรือไม่?

## สถานะเริ่มต้น
- สัญญาณไฟจราจร: วนลูปต่อเนื่อง — ไฟเขียว 15 วินาที → ไฟเหลือง 5 วินาที → ไฟแดง 10 วินาที (ไม่สามารถโต้ตอบได้ ต้องสังเกตเอง)
- ทางม้าลาย: ไม่มีไฟส่อง — อยู่บนถนนแต่มองเห็นเส้นขาวได้จางๆ
- ถนน: เปียกชื้นมันวาวจากละอองฝน — ผิวถนนลื่น
- ชายสูบบุหรี่: ยืนอยู่บนฟุตบาทฝั่งบ้าน ใกล้รั้ว — ยังไม่ได้โต้ตอบ
- ผู้หญิงฝั่งตรงข้าม: ยืนใต้เงามืดของไฟกะพริบ รอข้ามทางม้าลายอยู่ฝั่งตรงข้าม — ยังไม่ได้โต้ตอบ
- ถนน (ช่องทางนอกทางม้าลาย): เปิดให้ข้ามได้ทุกเมื่อ แต่อันตรายสูง

## ความผิดปกติในห้อง (Anomalies)

| ความผิดปกติ | พฤติกรรม | สถานะ |
|-------------|----------|-------|
| ไฟจราจรกะพริบสลับสี | สัญญาณไฟจราจรเปลี่ยนสีวนเอง เขียว → เหลือง → แดง → เขียว ไม่หยุด | วนลูปต่อเนื่องตลอดฉาก ไม่หยุดแม้ผู้เล่นจะโต้ตอบสิ่งอื่น |
| ผู้หญิงใต้เงามืดกะพริบ | ยืนนิ่งอยู่ฝั่งตรงข้าม แสงกะพริบทำให้ใบหน้าชัดบ้างหายบ้าง ดูเหมือนมองตรงมาที่ผู้เล่น | ยังคงอยู่จนกว่าผู้เล่นจะข้ามถนนหรือจบเกม |
| ไฟเสาถนนกะพริบ | ไฟเสาข้างทางม้าลายกะพริบไม่สม่ำเสมอ ทำให้ทางม้าลายปรากฏชัดบ้างหายบ้างในความมืด | ตลอดฉาก |
| คราบเลือดบนฟุตบาท (เงื่อนไขพิเศษ) | หากผู้เล่นเคยใช้มีดแล่ปลาทำร้ายชายสูบบุหรี่ — พื้นฟุตบาทจะมีร่องรอยนองเลือดอยู่เสมอ | ปรากฏถาวรตลอดเวลาที่อยู่ในฉาก ลด HP ต่อเนื่อง |

## วัตถุที่โต้ตอบได้

| วัตถุ | การกระทำ | เงื่อนไข | ผล |
|------|----------|----------|----|
| **สัญญาณไฟจราจร** | สังเกต | — | ผู้เล่นดูสีปัจจุบันของไฟจราจรได้ บันทึกสมุด ฯ อัตโนมัติ: *"ไฟจราจรยังคงใช้งานได้ รอจังหวะที่ปลอดภัยก่อนใช้ถนน"* (ไม่สามารถโต้ตอบโดยตรงได้) |
| **ชายสูบบุหรี่ (ทักทายอย่างสนิท)** | โต้ตอบ → เลือก "ทักทายอย่างสนิท" | — | ชายพยักหน้าตอบรับอย่างเย็นชา ไม่มีอะไรเกิดขึ้น — ผู้เล่นสามารถเดินต่อได้ |
| **ชายสูบบุหรี่ (ต่อว่าอย่างหยาบคาย)** | โต้ตอบ → เลือก "ต่อว่าอย่างหยาบคาย" | ไม่มีมีดแล่ปลาใน Inventory | ชายโกรธจัด — กระหน่ำทำร้ายผู้เล่นจนเสียชีวิต ☠️ |
| **ชายสูบบุหรี่ (ต่อว่าอย่างหยาบคาย)** | โต้ตอบ → เลือก "ต่อว่าอย่างหยาบคาย" | มีมีดแล่ปลาใน Inventory | มีปากเสียงกันเล็กน้อย — ผู้เล่นใช้มีดแล่ปลากระหน่ำแทงชายสูบบุหรี่จนนอนแน่นิ่งกับพื้น 🩹 (ลด HP ต่อเนื่องตลอดเวลาที่อยู่ในฉาก เนื่องจากอาการ Panic กำเริบจากการลงมือ) |
| **ทางม้าลาย + ข้ามถนน (ไฟแดง)** | เดินข้ามบนทางม้าลาย | ไฟจราจรเป็นสีแดง | ข้ามทางม้าลายได้สำเร็จ — พบผู้หญิงที่ปลายทางม้าลาย แสดงตัวเลือกโต้ตอบ |
| **ทางม้าลาย + ข้ามถนน (ไฟเหลือง)** | เดินข้ามบนทางม้าลาย | ไฟจราจรเป็นสีเหลือง | รถปรากฏในฉากพุ่งมา แต่เบรคทัน — ตกใจมาก 🩹 (ลด HP 0.50 หน่วย) ข้ามไปถึงปลายอย่างหวาดเสียว พบผู้หญิงที่ปลายทางม้าลาย แสดงตัวเลือกโต้ตอบ |
| **ทางม้าลาย + ข้ามถนน (ไฟเขียว)** | เดินข้ามบนทางม้าลาย | ไฟจราจรเป็นสีเขียว | รถพุ่งเข้ามาชนเต็มๆ ถนนลื่นเบรคไม่ทัน ☠️ |
| **ถนน (นอกทางม้าลาย) + ข้ามถนน (ไฟแดง)** | เดินข้ามถนนนอกทางม้าลาย | ไฟจราจรเป็นสีแดง | ข้ามถนนได้สำเร็จแต่ไม่ผ่านทางม้าลาย — ไม่สามารถโต้ตอบกับผู้หญิงได้ (ถือว่าเดินเลยไปแล้ว) ตัวละครเดินออกจากฉากไป ✅ จบเกม |
| **ถนน (นอกทางม้าลาย) + ข้ามถนน (ไฟเขียวหรือเหลือง)** | เดินข้ามถนนนอกทางม้าลาย | ไฟจราจรเป็นสีเขียวหรือสีเหลือง | รถปรากฏในฉากพุ่งเข้ามาชนอย่างรวดเร็ว ☠️ |
| **ผู้หญิงฝั่งตรงข้าม (ทักทาย)** | โต้ตอบ → เลือก "ทักทาย" | ข้ามทางม้าลายมาแล้ว (ไฟแดงหรือเหลือง) | ผู้หญิงยิ้มตอบรับเบาๆ — ตัวละครเดินผ่านออกไปจากฉาก ✅ จบเกม |
| **ผู้หญิงฝั่งตรงข้าม (ทำร้าย)** | โต้ตอบ → เลือก "ทำร้าย" | ข้ามทางม้าลายมาแล้ว + มีมีดแล่ปลาใน Inventory | ผู้เล่นใช้มีดแล่ปลาทำร้ายผู้หญิงจนเสียชีวิต — ตัวละครเดินออกจากฉากไปในความเงียบ ✅ จบเกม |
| **ผู้หญิงฝั่งตรงข้าม (ทำร้าย)** | โต้ตอบ → เลือก "ทำร้าย" | ข้ามทางม้าลายมาแล้ว + ไม่มีมีดแล่ปลาใน Inventory | ผู้หญิงกรี๊ดขอความช่วยเหลือ ผู้เล่นตกใจและถอยหลัง 🩹 (ลด HP 0.50 หน่วย) — ตัวละครวิ่งออกจากฉากไปอย่างรวดเร็ว ✅ จบเกม |

## Win Flow (ลำดับการเล่นที่ถูกต้อง)

```text
[เข้าสู่ฉากถนน — ยืนอยู่บนฟุตบาทฝั่งบ้าน]
    │
    ├─→ [ทางเลือกก่อนข้าม: โต้ตอบชายสูบบุหรี่ (ไม่บังคับ)]
    │        │
    │   ① คลิกชายสูบบุหรี่
    │        ├─→ เลือก "ทักทายอย่างสนิท"
    │        │        └─→ ไม่มีอะไรเกิดขึ้น — เดินต่อได้ปกติ
    │        │
    │        └─→ เลือก "ต่อว่าอย่างหยาบคาย"
    │                 ├─→ [มีมีดแล่ปลา] → มีปากเสียง → แทงชายสูบบุหรี่
    │                 │        └─→ 🩹 Panic กำเริบ ลด HP ต่อเนื่องตลอดฉาก
    │                 │
    │                 └─→ [ไม่มีมีดแล่ปลา] → ถูกทำร้ายจนตาย ☠️
    │
    ├─→ [WIN FLOW แบบที่ 1 — ข้ามทางม้าลาย + โต้ตอบผู้หญิง]
    │        │
    │   ① สังเกตสัญญาณไฟจราจรจนเป็นสีแดง (รอสูงสุด ~20 วินาที)
    │        │
    │   ② เดินข้ามบนทางม้าลายขณะไฟแดง
    │        └─→ ✅ ข้ามสำเร็จ — พบผู้หญิงที่ปลายทางม้าลาย
    │        │
    │   ③ แสดงตัวเลือกโต้ตอบผู้หญิง
    │        ├─→ เลือก "ทักทาย"
    │        │        └─→ ตัวละครเดินออกจากฉาก ✅ จบเกม
    │        │
    │        ├─→ เลือก "ทำร้าย" + [มีมีดแล่ปลา]
    │        │        └─→ ทำร้ายผู้หญิงจนตาย → ตัวละครเดินออกจากฉาก ✅ จบเกม
    │        │
    │        └─→ เลือก "ทำร้าย" + [ไม่มีมีดแล่ปลา]
    │                 └─→ ผู้หญิงกรี๊ด 🩹 ลด HP 0.50 → ตัวละครเดินออกจากฉาก ✅ จบเกม
    │
    └─→ [WIN FLOW แบบที่ 2 — ข้ามถนนนอกทางม้าลาย]
             │
        ① สังเกตสัญญาณไฟจราจรจนเป็นสีแดง
             │
        ② เดินข้ามถนนนอกทางม้าลายขณะไฟแดง
             └─→ ✅ ข้ามสำเร็จ — ไม่ผ่านทางม้าลาย ไม่สามารถโต้ตอบผู้หญิงได้
             │
        ③ ตัวละครเดินออกจากฉากไปโดยอัตโนมัติ ✅ จบเกม
```

> **หมายเหตุ:** ไม่ว่าจะเลือก win flow แบบไหน การโต้ตอบกับชายสูบบุหรี่ก่อนข้ามถนนไม่ส่งผลต่อการ win หรือ lose — แต่มีผลต่อ HP และบรรยากาศจบเกม ควรระวังหากเลือก "ต่อว่า" โดยไม่มีมีดแล่ปลา

## เบาะแส

| แหล่ง | เนื้อหา |
|-------|---------|
| สังเกตสัญญาณไฟจราจร | ไฟเปลี่ยนสีเป็นรอบ: เขียว 15 วินาที → เหลือง 5 วินาที → แดง 10 วินาที — ไฟแดงคือจังหวะปลอดภัยสำหรับข้ามทางม้าลาย |
| สังเกตถนน | ถนนมันวาวจากความชื้น — บ่งบอกว่ารถจะเบรคได้ยาก ต้องรอไฟแดงก่อนข้าม |
| สังเกตทางม้าลาย | ทางม้าลายไม่มีไฟส่อง — ข้ามทางม้าลายในความมืดขณะไฟเขียวจะไม่มีรถเห็นผู้เล่นทัน |
| สมุดบันทึกอัตโนมัติ | บันทึก: *"ไฟจราจรไกลๆ เปลี่ยนสีไปเรื่อยๆ — เขียว เหลือง แดง"* เมื่อสังเกตไฟจราจร |

## เงื่อนไขการตาย ☠️

| สถานการณ์ | เงื่อนไข | สาเหตุการตาย |
|-------------|----------|--------------|
| ถูกรถชนบนทางม้าลาย (ไฟเขียว) | เดินข้ามบนทางม้าลายขณะไฟจราจรเป็นสีเขียว | รถพุ่งเข้ามาชนเต็มๆ ถนนลื่นและบริเวณมืด รถเบรคไม่ทัน ตัวละครถูกชนกระเด็น |
| ถูกรถชนนอกทางม้าลาย (ไฟเขียว/เหลือง) | เดินข้ามถนนนอกทางม้าลายขณะไฟจราจรเป็นสีเขียวหรือสีเหลือง | รถปรากฏในฉากพุ่งเข้ามาอย่างรวดเร็ว ตัวละครถูกชนกระเด็นออกไปจากกรอบ |
| ถูกชายสูบบุหรี่ทำร้ายจนตาย | เลือก "ต่อว่าอย่างหยาบคาย" กับชายสูบบุหรี่ + ไม่มีมีดแล่ปลาใน Inventory | ชายโกรธจัด กระหน่ำทำร้ายผู้เล่นด้วยมือเปล่าจนเสียชีวิตบนฟุตบาท |

## เงื่อนไขการบาดเจ็บ 🩹

> [!WARNING]
> การบาดเจ็บสะสมอาจทำให้พลังชีวิต (HP) หมดและ Game Over ได้หากไม่ระวัง — ฉากนี้มีเงื่อนไขลด HP ต่อเนื่องหากก่อเหตุรุนแรงก่อนข้ามถนน ควรรีบจบเกมให้เร็วที่สุด

| สถานการณ์ | เงื่อนไข | ผลที่เกิด |
|-------------|----------|----------|
| รถเบรคทัน (ข้ามเหลือง) | เดินข้ามบนทางม้าลายขณะไฟจราจรเป็นสีเหลือง | รถพุ่งมาแต่เบรคทัน ตกใจมาก — ลด HP 0.50 หน่วยทันที ยังข้ามได้สำเร็จ |
| Panic จากการแทงชายสูบบุหรี่ | เลือก "ต่อว่า" + มีมีดแล่ปลา → แทงชายสูบบุหรี่ | อาการ Panic กำเริบจากการลงมือ — ลด HP ต่อเนื่องตลอดเวลาที่อยู่ในฉาก ต้องรีบจบเกมก่อน HP หมด |
| ผู้หญิงกรี๊ดขอความช่วยเหลือ | เลือก "ทำร้าย" ผู้หญิงฝั่งตรงข้าม + ไม่มีมีดแล่ปลาใน Inventory | ผู้หญิงกรี๊ดตกใจ ตัวละครตกใจและถอยหลัง — ลด HP 0.50 หน่วย ยังคงจบเกมหลังจากนั้นได้ |

## สิ่งที่เปลี่ยนเมื่อกลับมา
ฉากนี้ **ไม่มีการกลับมา** — เมื่อตัวละครเดินออกจากฉากไปสู่ฝั่งตรงข้ามสำเร็จ เกมจะจบลงทันที ไม่มีทางย้อนกลับมาจากฉากนี้ไปยังฉากก่อนหน้า

## Backtracking จากห้องอื่น
- ฉากนี้เป็นฉากสุดท้าย ไม่มี Backtracking เข้าหาจากห้องอื่น
- ผู้เล่นต้องมาจาก **รั้วหน้าบ้าน** เพียงทางเดียว และไม่มีทางกลับไปห้องก่อนหน้าได้อีก
- หากต้องการ **มีดแล่ปลา** เพื่อตัวเลือกพิเศษในฉากนี้ ต้องเก็บมาจากถังขยะที่ **รั้วหน้าบ้าน** ก่อนเข้าสู่ฉากถนน

## ไอเทมที่ได้จากห้องนี้
- ไม่มีไอเทมให้เก็บในฉากนี้ — ฉากนี้ใช้เฉพาะไอเทมที่มีอยู่แล้วใน Inventory เท่านั้น
- **มีดแล่ปลา** (จาก Inventory) → ใช้เป็นเงื่อนไขสำหรับตัวเลือกโต้ตอบกับชายสูบบุหรี่และผู้หญิงฝั่งตรงข้าม (Trigger — ไม่หายจาก Inventory เมื่อใช้)

## ทางออกจากห้อง

| ทางออก | สถานะ | เงื่อนไข |
|--------|-------|----------|
| ข้ามทางม้าลาย (ไฟแดง) | `✅ จบเกม` | ไฟจราจรเป็นสีแดง + เดินบนทางม้าลาย → พบผู้หญิง → โต้ตอบแล้วเดินออกจากฉาก |
| ข้ามถนนนอกทางม้าลาย (ไฟแดง) | `✅ จบเกม` | ไฟจราจรเป็นสีแดง + เดินนอกทางม้าลาย → ข้ามสำเร็จโดยไม่พบผู้หญิง → ออกจากฉาก |
| ข้ามทางม้าลาย (ไฟเหลือง) | `⚠️→✅` | ไฟจราจรเป็นสีเหลือง + บนทางม้าลาย → รถเบรคทัน 🩹 HP ลด 0.50 → พบผู้หญิง → จบเกม |
| ข้ามทางม้าลาย (ไฟเขียว) | `💀` | ถูกรถชนตาย |
| ข้ามถนนนอกทางม้าลาย (ไฟเขียว/เหลือง) | `💀` | ถูกรถชนตาย |
