# Horror Adventure — Game Design Document (ร่าง)

## ภาพรวมเกม

**ชื่อ (ชั่วคราว):** ตื่น (WAKE)
**แนว:** Point-and-Click Adventure / Psychological Horror
**มุมมอง:** First-person หรือ Third-person (2D หรือ 2.5D)
**แพลตฟอร์ม:** PC

---

## เรื่องย่อ

ตัวเอกตื่นขึ้นมาในห้องนอนของตัวเอง ความทรงจำเลือนราง รู้แค่ว่า "ต้องออกจากบ้านนี้"
บ้านดูปกติ — แต่ไม่ใช่
มีบางอย่างอยู่ในบ้าน มันจะไม่ขัดขวางตราบใดที่ทำทุกอย่างถูกต้อง
ผิดขั้นตอนเดียว — ตาย

---

## ห้องและลำดับ

```
ห้องนอน → ห้องน้ำ → โถงทางเดิน/บันได → ห้องครัว
→ ห้องทานข้าว → ห้องซักล้าง → ห้องนั่งเล่น
→ สวนหน้าบ้าน → รั้ว/ประตู → ถนน (จบเกม)
```

---

## แต่ละห้อง (สรุปคร่าวๆ)

### 1. ห้องนอน
- **วัตถุประสงค์:** ตื่นนอน หาเบาะแส เตรียมตัว
- **ปริศนา:** นาฬิกาปลุกดังผิดเวลา / ตู้เสื้อผ้าล็อกอยู่ / มีอะไรใต้เตียง
- **ตาย:** เปิดตู้ผิดบาน / มองใต้เตียงโดยไม่มีของป้องกัน

### 2. ห้องน้ำ
- **วัตถุประสงค์:** ล้างหน้า แปรงฟัน อาบน้ำ
- **ปริศนา:** กระจกสะท้อนภาพผิด / ยาในตู้มียาผิด / น้ำร้อน-เย็นสลับ
- **ตาย:** ใช้ยาผิดขวด / มองกระจกนานเกินไป / เปิดน้ำในลำดับผิด

### 3. โถงทางเดิน / บันได
- **วัตถุประสงค์:** ลงจากชั้นบนไปชั้นล่าง
- **ปริศนา:** บันไดบางขั้นห้ามเหยียบ / ภาพบนผนังบอกรหัส / เงาบนกำแพง
- **ตาย:** เหยียบขั้นที่ผิด / เดินผ่านภาพโดยไม่ทำตามสัญลักษณ์

### 4. ห้องครัว
- **วัตถุประสงค์:** ทำอาหารเช้า
- **ปริศนา:** เตาแก๊สต้องจุดในลำดับที่ถูก / ตู้เย็นมีของแปลก / มีดวางผิดที่
- **ตาย:** จุดเตาผิดลำดับ (ระเบิด) / เปิดตู้เย็นแล้วเห็นสิ่งที่ไม่ควรเห็น

### 5. ห้องทานข้าว
- **วัตถุประสงค์:** กินอาหารเช้า
- **ปริศนา:** นั่งผิดเก้าอี้ / รูปถ่ายครอบครัวบนผนังให้เบาะแส / โน้ตบนโต๊ะ
- **ตาย:** กินอาหารผิดจาน / นั่งเก้าอี้ผิดตัว

### 6. ห้องซักล้าง
- **วัตถุประสงค์:** เก็บของบางอย่างที่ต้องใช้
- **ปริศนา:** เครื่องซักผ้ากำลังหมุน มีของอยู่ข้างใน / สารเคมีผสมกัน
- **ตาย:** เปิดเครื่องซักผ้าตอนยังหมุนอยู่ / ผสมสารเคมีผิด

### 7. ห้องนั่งเล่น
- **วัตถุประสงค์:** หาของสำคัญ / ปลดล็อกประตูหน้า
- **ปริศนา:** ทีวีเปิดเองแสดงสัญลักษณ์ / หนังสือพิมพ์มีวันที่แปลก / กุญแจซ่อนอยู่
- **ตาย:** เปิดทีวีเองในเวลาผิด / หยิบกุญแจผิดดอก

### 8. สวนหน้าบ้าน
- **วัตถุประสงค์:** ข้ามสวนไปที่รั้ว
- **ปริศนา:** ดินในสวนบางจุดห้ามเหยียบ / จดหมายในตู้ไปรษณีย์ให้รหัส
- **ตาย:** เหยียบดินผิดจุด / เดินผ่านต้นไม้บางต้น

### 9. รั้ว / ประตู
- **วัตถุประสงค์:** ปลดล็อกประตูรั้ว
- **ปริศนา:** กุญแจรหัส 4 หลัก (รวบรวมจากเบาะแสทั้งเกม) / โซ่รั้วพันแบบพิเศษ
- **ตาย:** ใส่รหัสผิดเกิน 2 ครั้ง

### 10. ถนน (จบเกม)
- **วัตถุประสงค์:** ข้ามถนนออกไป
- **ปริศนา:** ข้ามถนนในช่วงเวลาที่กำหนด / มีสัญญาณบางอย่างต้องรอ
- **ตาย:** ข้ามผิดเวลา / วิ่งแทนที่จะเดิน

## กลไกหลัก

| กลไก | รายละเอียด |
|------|------------|
| **คลิกสำรวจ** | คลิกวัตถุเพื่อดูคำอธิบาย / รับของ / โต้ตอบ |
| **สมุดบันทึก** | เก็บเบาะแสและรหัสโดยอัตโนมัติ |
| **ตาย = รีเซต** | ตายแล้วเริ่มใหม่ที่ต้นห้อง พร้อมคำใบ้เพิ่มขึ้นเล็กน้อย |
| **ไม่มี Save Manual** | เกมเซฟอัตโนมัติเมื่อผ่านแต่ละห้อง |
| **ความเครียดสะสม** | เสียงและแสงเปลี่ยนตามจำนวนครั้งที่ตาย |

## โทนและบรรยากาศ

- เพลงประกอบ: Ambient drone, เสียงบ้านดังผิดปกติ
- สี: Desaturated, โทนเทา-น้ำเงิน, แสงจากหน้าต่างดูผิดเวลา
- การเล่าเรื่อง: ผ่านสภาพแวดล้อม ไม่มีตัวละครพูดโต้ตอบ

---

## ปริศนาหลักที่ร้อยเรื่อง

รหัสประตูรั้ว 4 หลัก ประกอบจาก:
- เลข 1: นาฬิกาในห้องนอน
- เลข 2: ยาในห้องน้ำ (เลขล็อต)
- เลข 3: วันที่ในหนังสือพิมพ์ห้องนั่งเล่น
- เลข 4: จดหมายในตู้ไปรษณีย์
